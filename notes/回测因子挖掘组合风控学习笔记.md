## 2. 因子挖掘索引

材料把因子挖掘分为：

- 基本面因子。
- 技术因子。
- 机器因子。
- 遗传规划因子挖掘。
- 神经网络因子挖掘。

理解：

```text
传统因子更重经济解释；
机器因子更重搜索空间和样本外验证；
LLM 因子更重语义生成、公式生成和审计。
```

因子挖掘最低要求：

- 明确输入字段。
- 明确公式。
- 明确方向。
- 明确数据可得时间。
- 明确样本内/样本外。
- 明确多重检验和过拟合控制。
- 明确与已有因子的相关性。

## 3. 组合优化与风控索引

材料给出三条入口：

- 凸优化。
- 投资组合优化实践。
- Barra 多因子风险模型。

理解：

组合优化不是“收益最大化”这么简单，而是在预测收益、风险模型、约束和交易成本之间求可执行解。

常见输入：

- 预期收益。
- 协方差矩阵。
- 风险暴露矩阵。
- 行业/风格约束。
- 个股/资产权重上下限。
- 换手约束。
- 成本模型。

常见风险：

- 协方差估计误差。
- 预期收益噪声。
- 约束过多导致组合不可行或过度集中。
- 优化器放大输入误差。
- 历史风险模型无法覆盖未来结构变化。

## 4. 海通动量/反转报告摘要

研究问题：

```text
A 股是否存在动量或反转效应？
```

研究设计：

- 股票池：剔除 ST 后所有 A 股。
- 每月月末按过去 J 个月收益率排序。
- 分成 q 组，报告中主要用 q=10。
- 分别买入收益最高组和收益最低组。
- 持有 K 个月。
- K>1 时采用多条资金链重叠持有。
- 与 Wind 全 A 指数比较。

核心发现：

- 高收益组在 A 股没有明显持续性，动量策略相对指数弱。
- 低收益组表现更好，反转效应强于动量效应。
- J=3、K=1 是报告中表现较优的反转参数。
- 分组收益显示一定单调性，低收益组未来更强。
- Alpha/Beta 分析显示低收益组 Alpha 更高，Beta 差异不大。
- 低估值 + 反转组合：中证 800 样本，先取 PB 较低股票，再选过去收益较低股票。

报告给出的可复现规则：

```text
股票池：中证800
调仓：每月最后一个交易日收盘后筛选
估值过滤：剔除 PB<0，保留 PB 最小前 25%
反转过滤：选其中过去收益率最低的 5%
成交：下月第一个交易日开盘买入
卖出：持有一个月至最后一个交易日收盘卖出
权重：等权
成本：双边 0.3%
基准：中证800
```

- 回测框架只是工具，不替代研究设计。

- 海通报告展示了完整因子研究表达：排序、分组、J/K 参数、信息比、单调性、Alpha/Beta、组合过滤和容量估算。

- 海通动量/反转研报复现计划。
- 绩效指标代码。

## 6. Falsification

主动反证：

- 海通报告样本期为 2007-2012，时间短，必须扩展到后续年份。
- 剔除 ST 不等于处理退市、停牌、新股、涨跌停和幸存者偏差。
- 高收益组无持续性不代表所有频率/市场状态下动量无效。
- 低收益组反转可能与小市值、流动性、跌停后反弹、行业暴露有关。
- 报告中的低估值 + 反转策略最大回撤仍很大，不能只看年化收益。
- 容量估算用成交额比例粗算，不能替代完整冲击成本模型。
- 研报结论不能直接升格为通用规则，必须复现。

## 7. Corrections

### Correction 1: 资料索引不是知识本体

原理解：

```text
读完 `资料/回测.md`、`资料/因子挖掘.md`、`资料/投资组合优化与风控.md` 就完成本主题。
```

问题：

```text
这三份文件只是链接和方向索引，内容不足以形成完整知识。
```

修正后：

```text
这些文件用于确定学习路径，真正知识来自课程模板、卖方研报、论文和代码复现。
```

影响：

```text
后续对索引型文件标记为 `index_read`，必须追踪至少一个下游材料。
```

### Correction 2: A 股“反转有效”不能无条件外推

原理解：

```text
A 股反转强于动量。
```

问题：

```text
该结论来自特定样本期、股票池、月度 J/K 参数和回测假设。
```

修正后：

```text
可记录为：海通 2012 报告在 2007-2012 A 股样本上发现月度低收益组反转强于高收益组动量，需扩展样本复现。
```

影响：

```text
因子库中反转因子标记为 `needs_replication`。
```

### Correction 3: 组合优化先审输入，不先审算法

原理解：

```text
组合优化核心是选择更高级算法。
```

问题：

```text
优化结果对预期收益、协方差和约束极敏感，输入噪声会被优化器放大。
```

修正后：

```text
组合优化先审预测、风险模型、约束和成本，再选择算法。
```

影响：

```text
后续组合模板必须包含 input_audit。
```
