## One-Line Takeaway

多因子模型不能把多个因子简单相加：因子之间的截面相关性会导致实际暴露偏离名义权重；正交化能让暴露更可控，但正交顺序、弱因子和输入噪音会显著影响效果。

## 1. 为什么需要因子正交

多因子模型中，因子之间常常相关，而且相关性不稳定。

问题：

```text
即使名义上给两个因子等权，组合实际可能因为因子相关性而对某个风格有更高暴露。
```

例子：

```text
市值因子和反转因子的长期平均相关性可能不高，
但在某些阶段可以显著正相关或负相关。
```

这会导致：

- 等权多因子模型实际暴露不可控。
- 权重优化模型更容易受相关性影响。
- 因子贡献归因变得困难。

## 2. 逐步正交方法

报告使用线性回归残差做正交。

设要把因子 `m` 对已有因子 `f1...fk` 正交：

```text
fm = beta1*f1 + beta2*f2 + ... + betak*fk + epsilon
orthogonal_fm = epsilon
```

逐步正交流程：

```text
1. 第一个因子保留原始值。
2. 第二个因子对第一个因子回归，取残差。
3. 第三个因子对前两个正交后因子回归，取残差。
4. 依次处理。
```

这类似 Gram-Schmidt。

## 3. 正交顺序

正交结果依赖顺序。

报告示例固定顺序：

```text
行业 -> 市值 -> 反转 -> 换手 -> 波动率 -> PB -> Beta -> 增速 -> 杠杆
```

含义：

- 先保留更基础、更强、更希望控制的暴露。
- 后加入因子只保留无法被前面因子解释的残差部分。

动态顺序方法：

```text
使用历史 24 个月收益预测模型 R 方选择下一个因子。
```

但报告也指出，R 方高不一定代表收益预测稳定，可能偏向风险因子。

## 4. 正交的效果边界

报告发现：

- 正交后复合因子 IC 未必大幅提升。
- 正交后 ICIR 更容易提升，说明稳定性改善。
- 在等权多因子模型下，正交提升会随弱因子增加而减弱。
- 在最大化收益预期模型下，弱因子正交后若贡献小，回归权重会降低，因此效果更稳定。

关键理解：

```text
正交主要改善暴露可控性和稳定性，不保证每次提高收益。
```

## 5. 因子加权与 Fama-MacBeth

第二篇报告给出一个重要性质：

```text
若收益率和因子都做 z-score，则最大化复合因子 IC 加权法等价于 Fama-MacBeth 回归。
```

直觉：

- 最大化 IC 是从“因子打分”角度建模。
- Fama-MacBeth 是从“收益率回归”角度建模。
- 在 z-score 条件下，两者可以互相转化。

实用意义：

```text
很多多因子加权问题可以转化成横截面回归问题，程序实现更直接。
```

## 6. 正交因子与因子溢价

报告性质：

```text
在 Fama-MacBeth 回归中，不论新因子是否与原有因子正交，其因子溢价估计不会改变。
```

实用意义：

```text
新因子可以先对已有模型因子正交，再加入模型。
这样既减少多重共线性，又不损失新因子与收益相关的信息。
```

## 7. 因子择时

报告把 Qian 因子择时模型解释为：

```text
拿因子溢价对条件变量集合建立回归模型。
```

关键风险：

- 条件变量太多会过拟合。
- 条件变量矩阵可能不满秩。
- 因子溢价本身噪音大。
- 线性假设可能不成立。

## 8. Review

这批材料解决的问题：

- 为什么多因子不能简单相加。
- 为什么因子相关性不稳定会污染实际暴露。
- 如何用回归残差做因子正交。
- 为什么正交顺序重要。
- 为什么因子加权、Fama-MacBeth 回归、因子择时可以放进统一回归框架。

与已有知识关系：

- 接上 Rank IC 因子评价。
- 为多因子合成、因子中性化、因子择时做准备。
- 连接组合风控中的风格暴露控制。

## 9. Falsification

主动反证：

- 正交化只能剔除线性相关，不能剔除非线性关系。
- 正交结果依赖因子顺序，不是唯一真相。
- 残差因子可能丢掉原始因子的经济含义。
- 弱因子正交后可能只剩噪音。
- 正交提升可能来自样本期和因子集合选择。
- 回归框架依赖线性假设。
- Fama-MacBeth 回归估计会受极端值、行业结构和异方差影响。

## 10. Corrections

### Correction 1: 等权多因子不等于等暴露

原理解：

```text
多个因子等权相加，就表示每个因子贡献相同。
```

问题：

```text
因子之间相关性会导致实际暴露偏离名义权重。
```

修正后：

```text
等权只是名义权重，实际暴露必须通过相关性、回归和组合归因检查。
```

影响：

```text
多因子模板必须输出因子相关性矩阵。
```

### Correction 2: 正交不是无损魔法

原理解：

```text
正交化一定让多因子模型更好。
```

问题：

```text
正交顺序、弱因子、样本期和线性假设都会影响结果。
```

修正后：

```text
正交是暴露控制工具，需要复现验证。收益提升不是必然。
```

影响：

```text
正交模板必须同时保留原始因子和正交因子，并比较两者表现。
```

### Correction 3: 因子择时是回归预测问题

原理解：

```text
因子择时是复杂独立模型。
```

问题：

```text
许多因子择时模型可转成因子溢价对条件变量的回归。
```

修正后：

```text
先用回归框架理解因子择时，再引入复杂模型。
```

影响：

```text
因子择时必须严格控制条件变量数量和样本外验证。
```
