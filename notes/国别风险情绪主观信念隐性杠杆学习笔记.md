## 核心收获

本轮把“投资者情绪”继续拆细。情绪不能只作为单一市场热度指标使用，至少应区分国别风险情绪、主观信念调整、理性/非理性情绪、估值状态、资金流和隐性杠杆约束。不同情绪变量对应不同资产定价通道，直接混合会导致解释混乱。

国别风险情绪影响跨境股票资本流动。全球投资者对某国的风险情绪上升，意味着该国风险溢价上升，跨境股票基金净流入下降。这个效应对风险厌恶更强的基金更明显，例如被动基金、开放式基金和 ETF。金融市场成熟度、汇率弹性可以缓和负面冲击；当全球风险情绪极低或某国权益基金流入极高时，影响更强。

主观信念调整是情绪影响波动的中间层。张宗新、王海亮的模型把信息偏好、贝叶斯信念更新和投资者情绪连接起来：基本面信息偏好更强，信念波动更低；噪声信息偏好更强，情绪波动更频繁。实证上可用 PCA 构建情绪指数，用股票型基金仓位代理机构信念，用股债波动率比代理主观风险溢价，再检验情绪、信念和市场收益/波动的动态关系。

估值状态改变情绪对波动的作用方向和主导者。胡昌生、池阳春将情绪分为理性情绪与非理性情绪：高估值阶段，非理性情绪对波动更显著；低估值阶段，理性情绪对波动更显著。理性投资者并不天然稳定市场，低估值阶段的理性交易也可能放大波动。

隐性杠杆约束把情绪、流动性和基金持仓风险暴露连接起来。主动权益基金无法直接加杠杆或做空时，会通过提高组合 beta 间接获得风险暴露。主动基金持仓聚合 beta 可以刻画隐性杠杆约束和资金流动性状态；在低情绪或低流动性状态下，聚合 beta 变化对股票和基金收益的条件定价更明显。

## 可量化变量

- 国别风险情绪：来自文本或新闻大数据的 country risk sentiment，使用滞后值解释国家/基金层面的权益资金流。
- 全球风险状态：VIX、全球风险情绪、美国影子利率、美元流动性等，用于控制全球共同冲击。
- 基金风险厌恶代理：被动/主动、开放/封闭、ETF/非 ETF、历史波动、赎回压力。
- 情绪 PCA 指标：封闭式基金折价率、换手率、新增开户数、上涨下跌家数比、市场 PE、指数振幅。
- 主观信念代理：股票型基金仓位、基金仓位变动、机构风险暴露变化。
- 主观风险溢价：股票指数月度波动率与债券指数月度波动率之比。
- 估值状态：市场 PE/PB 的历史分位数或中位数分组，高估值和低估值分别建模。
- 理性/非理性情绪：可用多情绪代理变量经 PCA/因子分析后分离，也可用与基本面正交后的残差情绪近似非理性部分。
- 隐性杠杆约束：主动权益基金持仓加权 beta 聚合值及其变化量。
- 条件定价状态：低情绪、低流动性、高基金流入、高风险厌恶、不可融资融券等状态。

## 研究设计

跨境资本流：

`flow_{fund,country,t+1} = a + b * risk_sentiment_{country,t} + controls + fund FE + country FE + time FE + e`

重点检验 `risk_sentiment` 与基金类型、金融市场成熟度、汇率弹性、全球风险状态的交互项。需要避免把过去收益导致的赎回/申购误认成情绪冲击，因此必须加入过去市场收益、汇率变化、利率、通胀和宏观增长。

主观信念与市场波动：

`return_or_vol_{t+1} = a + b1 * sentiment_t + b2 * fund_position_t + b3 * subjective_risk_premium_t + macro_controls_t + e`

建议同时做 VAR/脉冲响应或分布滞后模型，确认情绪冲击的持续期。情绪和基金仓位不能用不可交易时点的数据。

估值条件下的情绪波动：

`vol_{t+1} = a + b1 * rational_sentiment_t + b2 * irrational_sentiment_t + controls + e`

在高估值和低估值状态分别估计，或加入情绪与估值状态交互项。还应区分加权波动率和等权波动率，检验大盘股与小盘股差异。

隐性杠杆约束：

`aggregate_fund_beta_t = sum_fund(size_weight_fund,t * sum_stock(weight_stock,fund,t * beta_stock,t))`

用 `delta_aggregate_fund_beta` 表示约束变化，再估计股票或基金对该变量的暴露。收益检验必须条件化：低情绪/低流动性时才更可能出现显著风险补偿。

## 复习与纠错

情绪指数不能包含被解释变量的近似版本。例如用市场 PE 或指数振幅构建情绪，再解释估值或波动，会产生机械相关。解决方式是使用滞后、正交化，或构建剔除估值/波动代理后的情绪版本。

跨境资金流存在强反向因果。过去收益、汇率升值和宏观改善会同时改变资金流和文本情绪，因此必须使用滞后情绪、固定效应和充分控制变量，必要时做事件窗口或工具变量设计。

基金持仓 beta 的频率通常低于市场数据。半年报/季报持仓会有窗口粉饰和披露滞后，不能把披露日之后才可获得的信息放到历史决策点之前。

“理性情绪”不是稳定器。低估值阶段的理性资金可能因为估值修复、仓位调整或流动性约束而制造波动，不能把理性投资者默认当作套利纠偏力量。

隐性杠杆约束不是无条件收益因子。只有在低情绪、低流动性、融资约束强或不可卖空状态下，暴露才更可能被定价。全样本直接排序可能掩盖机制。
