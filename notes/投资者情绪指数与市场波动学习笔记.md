## 在线社区支持倾向

钱宇等使用东方财富网 5,178,824 条评论，通过 CNN 分类识别看涨/看跌支持倾向，并区分“支持倾向”和“情绪”：

- 支持倾向：用户对未来市场看涨或看跌的预期，更接近投资行为。
- 情绪：乐观或悲观的朴素情感。

指标构造：

```text
S_t = (Bull_t - af * Bear_t) / (Bull_t + af * Bear_t) * log(1 + Bull_t + Bear_t)
```

`af` 用于处理正负信息非对称。意见一致性使用 bullish 与 bearish 数量差异构造，衡量群体共识程度。

核心结论：

- 当期支持倾向对未来市场收益显著为负。
- 支持倾向一致性会放大市场波动。
- 用户支持倾向很大程度基于历史市场表现形成，存在羊群效应。

## 市场情绪指数与价值偏离

丁志国等从噪声交易、信息成本和价值回归讨论市场情绪指数假说：

- 价格不能完全及时反映信息，存在价值偏离。
- 信息冲击后价格会向内在价值回归，但偏离不会立刻消失。
- 市场情绪影响投资者对信息持续性和内在价值的估计。

量化启发：

- 情绪指数不只预测收益，还可能改变投资者对价值的估计路径。
- 估值偏离、信息反映周期和情绪状态可以联合建模。

## 情绪与时变风险补偿

贺志芳与王美今等的框架把情绪变化放进收益均值和波动方程：

- 情绪变化影响均衡收益。
- 情绪变化影响风险或条件方差。
- 情绪可能通过风险补偿渠道影响收益。
- 市场弱式有效不能只用白噪声检验，还要看条件期望中的非线性依赖。

典型模型：

```text
R_t = alpha_0 + alpha_1 * h_t + alpha_2 * DeltaSI_t + error_t
h_t = GARCH/GJR terms + sentiment terms
```

## 情绪波动与资产价格波动

陈彦斌把情绪波动定义为投资者主观偏好参数的波动：

- 主观贴现因子。
- 风险规避系数。
- 跨期替代弹性。

基于 Epstein-Zin 递归效用，论文区分风险规避和跨期替代弹性，研究主观参数局部波动对股票和债券价格的影响。

主要启发：

- 情绪不是只能来自文本或调查，也可以建模为偏好参数的随机波动。
- 股票价格对情绪参数波动的敏感性明显大于债券。
- 主观贴现因子、跨期替代弹性、风险规避都可能是资产价格波动源。

## 研究纠错

- 情绪指标必须明确是 mood、support、attention、agreement 还是 flow。
- 当日评论和当日收益存在同步性，预测回归必须处理可得时点和滞后。
- 情绪平均值和意见一致性含义不同：前者影响方向，后者影响波动和羊群。
- 文本模型精度不是终点，最终要验证金融变量上的边际解释力。
- GARCH/TGARCH 结果不能直接转化为交易策略，需要样本外和成本验证。
