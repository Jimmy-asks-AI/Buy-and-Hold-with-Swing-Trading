# 指数级行业轮动与大小盘切换基础模型设计

日期：2026-05-24

## 目标

用当前已经落地的指数数据，先构建一个可回测、可解释、低频调仓的基础模型：

1. 判断大盘、中盘、小盘哪种市值风格占优。
2. 在行业指数之间做行业轮动。
3. 最终输出一组指数或ETF代理权重。

该模型只使用指数级数据，不使用历史成分股聚合，避免当前数据不足带来的未来函数。

## 可用数据

行业轮动：

- 申万行业指数日线：`data_raw/index/akshare_sw_industry/daily_sw/`
- 申万行业分类：`data_raw/index/akshare_sw_industry/classification/`
- 当前行业估值快照：`data_raw/index/akshare_sw_industry/analysis_latest_sw/`

大小盘切换：

- 沪深300：`000300`
- 中证500：`000905`
- 中证1000：`000852`
- 中证全指：`000985`
- 指数日线：`data_raw/index/akshare_csindex/daily_csindex/`
- PE/PB：`data_raw/index/akshare_csindex/valuation_pe_lg/`、`valuation_pb_lg/`

## 模型结构

### 第一层：市值风格切换

候选资产：

- 大盘：沪深300 `000300`
- 中盘：中证500 `000905`
- 小盘：中证1000 `000852`

核心信号：

- 相对动量：过去 60 / 120 日收益率。
- 相对趋势：指数价格是否高于 120 日或 200 日均线。
- 相对估值：PE/PB 历史分位和大小盘估值 spread。
- 风险惩罚：过去 60 日波动率、最大回撤。

基础得分：

```text
style_score =
  0.35 * zscore(120日收益率)
+ 0.25 * zscore(60日收益率)
+ 0.20 * zscore(趋势强度)
+ 0.10 * zscore(估值吸引力)
- 0.10 * zscore(波动率)
```

输出：

- 选择得分最高的 1 个风格，或选择前 2 个风格按得分加权。
- 加入换手约束：只有新风格得分比旧风格高出阈值时才切换。
- 建议调仓频率：月频，周频仅用于风控降仓。

### 第二层：行业轮动

候选资产：

- 默认用申万一级行业 31 个。
- 若需要增强主题表达，可以加入重点二级行业，但要避免一级和二级重复暴露。

核心信号：

- 行业相对强弱：行业指数过去 60 / 120 日相对中证全指的超额收益。
- 行业趋势：价格相对 120 日均线、均线斜率。
- 行业短期过热惩罚：20 日涨幅过高、偏离均线过大。
- 行业风险惩罚：60 日波动率、最大回撤。
- 行业估值：当前 PE/PB/股息率仅用于当前截面判断，不用于历史回测。

基础得分：

```text
industry_score =
  0.35 * zscore(120日相对收益)
+ 0.25 * zscore(60日相对收益)
+ 0.20 * zscore(趋势强度)
- 0.10 * zscore(短期过热)
- 0.10 * zscore(波动率)
```

输出：

- 选择前 3-5 个行业。
- 单行业权重上限 25%-35%。
- 若行业总趋势为负，可保留现金或切回宽基指数。

### 第三层：组合合成

组合权重可以分两块：

```text
最终组合 = 50% 市值风格指数 + 50% 行业轮动篮子
```

更稳健的版本：

```text
牛市/趋势强：40% 市值风格 + 60% 行业轮动
震荡/弱趋势：70% 市值风格 + 30% 行业轮动
熊市/风险高：宽基低仓位或现金代理
```

## 风控规则

- 指数低于 200 日均线且 60 日收益为负：降低该指数权重。
- 组合最大行业数：5 个。
- 单行业最大权重：30%。
- 单次调仓最大换手：50%。
- 连续回撤超过阈值：降低行业轮动权重。
- 成交成本假设：指数/ETF单边 5-20bp，回测至少做 5bp、10bp、20bp 三档敏感性。

## 回测规则

- 信号日：每月最后一个交易日收盘后计算。
- 执行日：下一个交易日开盘或收盘执行。
- 避免未来函数：所有滚动指标只用信号日前数据。
- 样本切分：训练/参数选择和测试分离。
- 输出指标：年化收益、波动率、Sharpe、Sortino、最大回撤、胜率、换手率、成本后收益、分年度收益。

## 当前不能做的部分

- 不能把当前行业成分权重用于历史成分股聚合。
- 不能用当前行业估值快照做历史估值分位回测。
- 不能声称是股票级大小盘因子，只能称为大小盘指数代理。

## 最小可行版本

第一版只做价格信号：

1. 大小盘指数：沪深300、中证500、中证1000。
2. 行业指数：申万一级行业。
3. 信号：60日收益、120日收益、120日均线趋势、60日波动率。
4. 调仓：月频。
5. 组合：50% 风格指数 + 50% 行业前5。

这版数据当前已经足够，可以直接进入代码实现和回测。
