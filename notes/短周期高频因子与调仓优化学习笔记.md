## 核心问题

本轮从“高频因子是否有效”转到“高频因子如何进入低频组合”。核心结论：

- 高频因子的信息半衰期短，不应简单通过提高调仓频率获取收益。
- 更稳健的用法是：在原低频组合框架下，用高频信号微调买卖节奏、过滤短端风险、提升收益预测模型的边际信息。
- 高频因子必须和成交成本、滑点、冲击、涨跌停可交易性一起评估，否则很容易把毛收益误判为可交易收益。

## 直观逻辑低频化

报告把不同层级高频数据统一到分钟级：

- 分钟 K 线：成交额、收益率、成交笔数。
- 盘口委托快照：净委买变化金额。
- 逐笔成交：主动买入、主动卖出、净主动买入、大买成交金额。

核心买入意愿指标：

```text
buy_intention = net_bid_change_amount + net_active_buy_amount
buy_intention_ratio = sum(buy_intention) / sum(amount)
buy_intention_intensity = mean(buy_intention) / std(buy_intention)
```

常用时段：

- 全天：9:30-14:56。
- 开盘：9:30-9:59。
- 盘中：10:00-14:26。
- 收盘：14:27-14:56。

报告中，开盘买入意愿强度正交后表现较强，月均 IC 约 0.04，ICIR 约 3.70，月度胜率约 85%，月均多空约 1.34%。买入意愿相对单纯净主动买入更强，说明“未成交挂单意愿 + 已成交主动买入”比单一成交数据更完整。

## 机器学习发现高频表达式

机器学习模块不是直接端到端预测收益，而是在高频基础变量和算子空间中搜索表达式。数据层级包括：

- 分钟成交额、收益率。
- 盘口净委买变化金额。
- 主动买入、净主动买入、大买成交金额。

常用算子包括：

- 四则运算、取负、倒数、对数。
- 日内时段切片：开盘、盘中、收盘、全天。
- 滚动统计：20 日 rolling sum/mean/std。

代表性表达式：

```text
Alpha1 = -log(rolling_std(period_mid(active_buy_amount)))
Alpha2 = -log(rolling_std(big_buy_amount * period_open(amount)))
Alpha4 = -log(rolling_mean(amount - period_mid(amount)))
Alpha6 = -rolling_sum(period_open(amount) + period_close(amount)) / rolling_sum(period_mid(amount))
```

经验解释：

- 主动买入波动越大，越可能是短期噪声或冲击，不一定利好。
- 开盘和收盘成交异常放大可能代表投机或被动交易压力，后续反而弱。
- 机器学习挖掘出的因子仍需要经济解释、稳定性、相关性和交易成本验证。

## 短周期因子与延迟调仓

高频因子在短持有期更强，但直接提高换手会带来成本问题。报告提出的实用机制是“延迟调仓”：

1. 低频模型在调仓日 T 生成原始买入和卖出列表。
2. 用短周期高频因子给计划卖出股票和计划买入股票排序。
3. 对计划卖出中短期信号最强的一部分，延迟卖出。
4. 对计划买入中短期信号最弱的一部分，延迟买入。
5. 在 T+N 再执行延迟的卖出和买入。

参数：

- `M`：延迟交易比例或延迟仓位比例。
- `N`：延迟天数。

约束：

- `M` 太低，组合影响小；太高，信号组间差异被稀释。
- `N` 太短，收益空间小；太长，超过高频信号半衰期。
- 计划买入和计划卖出最好保持金额平衡，否则会引入现金暴露或杠杆变化。

报告中，复合高频因子比单因子更稳健；对中证 500 的增强更明显，年化收益提升约 1%-2%；对沪深 300 提升约 0.5%。这说明高频信号的组合价值受股票池、流动性和基础组合换手制约。

## 高频因子的现实约束

高频数据并非越细越好。关键约束包括：

- 上交所和深交所 Level2 字段不同，逐笔委托和撤单可用性不一致。
- 盘口挂单可撤，净委买变化可能被虚假挂单污染。
- 高频因子多数持有期短，成本和滑点对收益侵蚀很大。
- 很多因子收益来自空头端，A 股多头组合不能直接照搬多空收益。
- 部分研报使用调仓日均价或 VWAP，真实执行不一定可获得同样价格。
- 报告中组合测试可能扣除单边 0.3% 成本，但仍未完整覆盖冲击成本、容量和涨跌停无法成交。

## 高频因子的组合用法

1. 直接加入收益预测模型。
   - 适合有成熟多因子指数增强框架的组合。
   - 需要验证新增因子对原模型的边际 IC、边际多头收益和约束后收益。

2. 只作为空头过滤。
   - 适合收益主要来自空头端的高频因子。
   - 不把信号当正向 alpha，而是排除或降权短期风险最高的持仓。

3. 后置剔除短端股票。
   - 先完成组合优化，再剔除高频信号最差的少数股票并重分配权重。
   - 报告中后置剔除通常比前置剔除更直接，因为它保留了原模型的完整优化信息。

4. 延迟调仓。
   - 不改变基础调仓频率。
   - 只改变部分计划交易的执行时点。
   - 更适合“低频模型 + 短周期微观信号”的组合。

## 复习与纠错

- 纠错 1：高频因子 IC 高不等于可交易。必须扣除手续费、印花税、滑点、冲击和不可成交约束。
- 纠错 2：提高调仓频率不是默认更优。毛收益上升可能完全被成本吞噬。
- 纠错 3：空头端有效的因子不能直接当多头买入信号。它更适合排除、降权或延迟买入。
- 纠错 4：调仓优化依赖基础组合的计划交易池。基础换手太低时，可延迟的交易太少，收益增强空间会变小。
- 纠错 5：机器学习表达式必须经过经济解释和样本外稳定性验证，不能只因为回测 IC 高就入库。
