## 1. 因子季节效应

报告用 2005.07-2022.06 的月度因子溢价研究月历效应和假日效应。因子先去极值、标准化；除市值外，其余因子对市值和行业正交。月溢价来自股票收益对因子暴露的截面回归。

全样本结论：

- A股长期呈小盘价值风格，但市值和估值因子波动大，月胜率只有约 50%。
- 低涨幅、低波动、低换手、高ROE、高SUE更稳定，年胜率约 90%-95%。
- 小市值与反转因子正相关；低估值与低波动正相关，与ROE、SUE负相关。价值风格下低波更好，成长风格下盈利和增速更好。

月历效应：

- 市值：2、3月小盘最强，年化单位溢价约 19%，胜率超过 80%；5、8月也偏小盘。6、12月小盘走弱，大盘相对更好。解释是春节风险偏好和3/5/8月资金宽松，半年末资金压力推高大盘流动性需求。
- 估值：5、6月显著偏成长，估值因子溢价为正，稳健回归年化约 11%；其他月份偏价值，年化约 -3.5%。解释是年报和一季报披露后，成长属性得到验证。
- 价量：反转、低波、低换手下半年强于上半年。换手率因子上半年年化约 -5.9%，下半年约 -10.9%。解释是上半年主题和热点延续性更强，下半年情绪更谨慎。
- 基本面：ROE/SUE在岁末年初、年报/一季报后、中报后表现更好。5-6、9-10、12-1月虚拟变量下，ROE年化单位溢价约 8.4%，SUE约 6.5%。

假日效应：

- 节前市场追逐确定性，大市值、低换手、高ROE表现突出。
- 节后市场追求弹性，小市值、高估值、反转表现突出。
- 策略示例：假期前 5 日配置高盈利组合，平均相对 Wind 全A 超额约 0.75%；假期后 5 日配置小市值+高增长组合，平均相对超额约 1.55%。
- 风控应用：2、3、5、8月放松市值暴露约束，沪深300增强年化超额从 10.7% 提升至 14.1%；5、6月放松估值成长暴露，中证500增强年化超额从 20.4% 提升至 21.0%。

研究判断：

- 季节效应不应作为单独交易系统，更适合作为卫星策略开关、风格暴露上限调整、因子权重小幅倾斜。
- 必须滚动复核，因为季节效应容易被公开后衰减。

## 2. 买卖单主动成交度

逐笔成交有主动方向，但一个买单或卖单本身可能同时包含主动与被动成交。报告从买卖单视角重构订单，并计算不同订单大小的主动成交度。

定义：

- 大买/卖单主动成交度 = 大买/卖单主动成交金额 / 大买/卖单成交金额
- 中买/卖单主动成交度 = 中买/卖单主动成交金额 / 中买/卖单成交金额
- 小买/卖单主动成交度 = 小买/卖单主动成交金额 / 小买/卖单成交金额
- 大/中/小阈值基于个股 20 日订单成交额分布滚动计算：大单为大于均值+1倍标准差，中单为大于均值且小于等于均值+1倍标准差，小单为小于均值。

核心发现：

- 大单和中单主动成交度月度选股能力不显著，小单主动成交度最强。
- 小买单主动成交度月均IC约 0.050，ICIR约 2.00，月胜率约 74%，月均多空约 1.78%，月均多头超额约 0.80%。
- 小卖单主动成交度同样为正向因子，说明“小单主动性越强”本身可能代表活跃且分歧被消化，而不只是买入方向。
- 盘中小买单主动成交度强于开盘后，月均IC约 0.053，ICIR约 2.11。
- 对行业、市值、换手、反转正交后，小买单主动成交度仍显著：全天/盘中月均IC约 0.032/0.033，ICIR均约 2.41，月胜率约 76%-77%。
- 在中证800外更强，正交后不同股票池差距缩小。

组合应用：

- 如果基础模型不含深度学习高频因子，加入小买单主动成交度可将中证500/1000增强年化超额提高约 0.5%-1.5% 或 0.5%-2.0%。
- 如果基础模型已含深度学习高频因子，增量不稳定，说明信息可能已被更高维高频模型吸收。

研究判断：

- 小单主动成交度是可入库高频因子，但进入多因子模型前必须检查与买入意愿、大单净买入、深度学习高频因子的边际贡献。
- 它更适合作为中小盘/高频模型增量，而不是取代传统大单资金流。

## 3. 深度学习高频因子特征工程

报告将深度学习高频因子的特征工程拆为四步：特征构建、特征处理、特征归因、特征筛选。

特征构建：

- 高频数据分为分钟K线、3秒盘口快照与委托队列、逐笔成交与逐笔委托。
- 实操上可将不同频率统一降频至分钟级，再构建目标频率特征。
- 推荐流程：原始数据 -> 分钟级基础指标 -> 目标频率衍生指标。
- 报告使用“固定算子、变换输入”的方式：先从人工逻辑高频因子中提炼算子，再替换输入序列，生成大量特征。
- 176个30分钟特征集合训练出的深度学习因子周均IC约 0.072，年化ICIR约 8.884，周胜率约 90%，Top10%多头年化超额约 28.5%。

特征处理：

- 偏度调整：对收益波动率、成交金额、成交笔数、买卖单数等偏度大的特征做对数/分布调整。176特征集合多头年化超额从 28.5% 提升到 31.1%。
- 去极值：在偏度调整后再做 N 倍标准差截断，多头年化超额提升到约 31.4%，ICIR约 9.265。
- 标准化：单一截面标准化与跨截面标准化差异不大。跨截面标准化保留一部分时间序列信息，但没有稳定显著优于截面标准化。

特征归因：

- 归因方法分为梯度类和扰动类。报告选用积分梯度 IG，因为它有完整性：归因值之和可分解模型输出与基线输出差。
- 静态按全样本平均贡献度选 Top64 特征会引入未来信息，只能做研究诊断。
- 动态特征筛选更合理：每次先用176特征训练，计算贡献度，再选贡献度最高的128/64/32个特征重新训练。

特征筛选：

- 特征从176精简到128或64后，IC通常不下降，多头超额进一步提升。
- 精简到32可能过度，导致有效信息损失。
- 动态64特征集合在多数处理方式下优于原始176特征，是较好的复杂度/效果折中。

组合应用：

- 中证500增强中，特征预处理和动态特征筛选多数情况下提升年化超额；示例参数下 2016-2022年化超额约 17.8%，月胜率约 86%。
- 中证1000增强中，效果更强；示例参数下 2016-2022年化超额约 24.3%，月胜率约 88%。

## 4. 多颗粒度深度学习模型

单颗粒度结果显示，日度特征平均表现不弱，但60分钟和30分钟特征在部分年份更优，说明细颗粒度有增量信息。

两种融合方式：

- 混合输入：不同颗粒度特征分别通过 GRU 提取序列信息，再合并输出。
- 输出集成：每个颗粒度单独训练模型，最终对预测输出做简单平均。报告中输出集成整体效果最好。

关键结论：

- 多颗粒度模型在不同标签长度、调仓周期、成交价格假设下，Rank IC普遍高于单颗粒度。
- 输出集成的10日标签 Top10%费前年化超额约 31.5%。
- 为缓解 RNN 遗忘问题，引入注意力机制并把 GRU 改为双向，形成双向AGRU。
- 双向AGRU多颗粒度模型周均Rank IC超过0.12，Top10%与Top100组合费前年化超额约 33%和40%；扣双边0.3%后仍约 24%和30%。
- 模型在中证800内表现一般，在中证1000、国证2000、中证1800外、国证3000外等中小盘域更强。2023年中证500/1000/国证2000内表现低于历史均值，沪深300内反而更强，可能反映拥挤迁移。
- 多颗粒度残差学习网络复现效果弱于输出集成，复杂模型不必然更优。

AI增强组合：

- 无成分股约束时，中证500 AI增强年化超额约 15%-20%，中证1000约 25%-30%。
- 加入80%成分股权重约束后，中证500降至约10%-15%，中证1000降至约22%-28%。成分股约束对500影响更大。

## 复习与纠错

- `concept_fix`：季节效应不是稳定交易规律，只能做滚动验证后的风格倾斜。
- `method_fix`：主动成交度必须从订单级重构，不能把逐笔BS方向直接等同于买单/卖单主动性。
- `method_fix`：深度学习特征归因用于筛选时，必须使用滚动历史贡献度；全样本静态贡献度只能诊断，不能回测。
- `method_fix`：高频深度学习因子必须单独评估与传统高频因子的边际贡献，避免重复计量。
- `data_fix`：2023-05-14 PDF 文本层损坏，本轮不标记为已学习，后续需要 OCR 或重新获取。
- `code_fix`：新增季节效应工具、机器学习特征工程工具，并在订单流模块新增买卖单主动成交度计算函数。
