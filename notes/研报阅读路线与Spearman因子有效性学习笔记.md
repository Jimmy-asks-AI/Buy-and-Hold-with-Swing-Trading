## One-Line Takeaway

读卖方金工研报先抓研究结构，不先啃公式；Spearman/Rank IC 的核心意义是用横截面排序关系刻画因子与未来收益的关系，但单期 Rank IC 噪音很大，必须看时间序列稳定性和样本外预测。

## 1. 研报阅读路线

第一遍读研报，只抓：

- 研究问题。
- 研究对象。
- 数据范围。
- 因子或策略定义。
- 评价方法。
- 主要结论。
- 风险和局限。
- 能迁移到自己研究的问题。
- 暂时看不懂的部分。

先看：

```text
标题 -> 摘要 -> 目录 -> 图表 -> 结论 -> 方法细节
```

暂时跳过：

- 复杂公式推导。
- 过长文献综述。
- 附录。
- 暂时不用的机器学习细节。

## 2. 海通 Spearman 报告研究问题

报告问题：

```text
如何从因子与股票收益的 Spearman 相关系数出发，判断因子有效性，并动态选择因子？
```

它关心两个问题：

- 数据噪音：单期相关系数波动很大，可能只是噪音。
- 数据时效性：长窗口稳定但滞后，短窗口及时但噪音更大。

## 3. 因子打分选股框架

报告中的基本框架：

- 股票池：每期沪深 300 成分股。
- 因子池：常见成长、估值等因子。
- 投资周期：月度，起始为自然月最后一个交易日，终止为下个自然月最后一个交易日。
- 因子打分：
  1. 确定因子和打分方向。
  2. 按因子表现给股票打分。
  3. 多因子分数等权相加。

重要边界：

```text
因子选择方法的好坏，不应只由最终组合表现判断，因为组合表现还受权重、约束、行业中性和交易规则影响。
```

## 4. Spearman / Rank IC 定义

报告中的 Spearman 相关系数是截面相关系数：

```text
在投资周期起始日 t，读取股票池中每只股票的因子值；
读取 t 之后一个月的每只股票收益率；
剔除缺失和异常；
计算因子值与次月收益率之间的 Spearman 相关系数。
```

在常见因子研究语境中，它对应 Rank IC。

直觉：

```text
因子排序和未来收益排序越一致，Rank IC 越高。
```

## 5. 有效因子的新理解

报告提出：

```text
观测到的 Spearman 相关系数 = 真实相关性 + 噪音相关性
```

结论：

- 单期 Spearman/Rank IC 波动很大。
- 噪音占比可能很高。
- 因子真实有效性更像一个随时间变化、但具有一定延续性的隐含状态。
- 因子有效性不应只看当期观测值，也不能只看过长窗口均值。

## 6. p 值法和移动平均

报告解释：

```text
p 值因子选取法本质上近似于 24 个月移动平均模型。
```

优点：

- 稳定。
- 简单。

缺点：

- 滞后。
- 如果均值在阈值附近震荡，因子选取不稳定。
- 对市场风格变化反应慢。

## 7. Kalman Filter 的角色

报告使用马尔科夫链 Kalman Filter 估计因子“真实”相关性。

这里先不把重点放在公式，而放在研究思想：

```text
把 Rank IC 看作带噪音的观测值；
把真实因子有效性看作需要估计的隐状态；
用模型在稳定性和时效性之间折中。
```

这比“直接拿最近一期 IC 决定因子是否有效”更合理。

## 8. 组合实证框架

报告实证：

- 股票池：沪深 300。
- 投资周期：月度。
- 因子选择：KF 法、p 值法、全因子法。
- 组合构建：行业中性，选 50 只股票，等权。
- 对冲：以沪深 300 指数为基准。
- 评价：年化超额收益、年化波动、信息比、最大回撤。

报告发现：

```text
其样本内 KF 法表现优于 p 值法和全因子法。
```

但该结论必须复现后才能采用。

## 10. Falsification

主动反证：

- 报告样本期为 2008-2013 附近，样本不长。
- 组合表现受行业中性、选股数量、等权、对冲假设影响，不能只归因于因子选择方法。
- 沪深 300 股指期货可对冲不等于所有投资者可实现。
- KF 模型可能也有参数设定和样本内优化问题。
- Rank IC 序列预测因子未来有效性，需要严格样本外验证。
- 因子池若经过事后筛选，会有数据挖掘偏差。

## 11. Corrections

### Correction 1: Rank IC 不是稳定真相

原理解：

```text
Rank IC 是因子有效性的直接度量。
```

问题：

```text
单期 Rank IC 噪音很大，可能主要反映随机扰动。
```

修正后：

```text
Rank IC 是带噪声观测值，需要看滚动均值、稳定性、样本外和市场状态。
```

影响：

```text
因子评价模板必须输出 Rank IC 时间序列和滚动统计。
```

### Correction 2: 因子选择和组合构建要分开评价

原理解：

```text
组合表现好说明因子选择方法好。
```

问题：

```text
组合表现还受行业中性、选股数量、权重、对冲和成本影响。
```

修正后：

```text
先评价因子选择对未来 Rank IC 的预测能力，再评价组合结果。
```

影响：

```text
复现报告需分为 factor_selection_evaluation 和 portfolio_backtest_evaluation。
```

### Correction 3: 研报阅读不能从公式开始

原理解：

```text
读金工研报要先看懂公式。
```

问题：

```text
先啃公式容易丢失研究问题、数据和评价逻辑。
```

修正后：

```text
先抽取研究结构，再回头处理公式。
```

影响：

```text
后续所有 PDF 先填研报拆解表。
```
