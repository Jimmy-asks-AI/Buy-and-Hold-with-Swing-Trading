# Task Briefs

This directory stores machine-readable task briefs for governed subagent work.

Historical tasks before `20260527_subagent_effectiveness_critique_v4` were often recorded only in `reports/AGENT_TASK_BOARD.md`. Future governed tasks should add one JSON brief here before acceptance.

Each brief must validate against `strategy_lab/agents/_templates/task_brief.schema.json` and must keep context scoped to explicit `allowed_inputs` and `allowed_writes`.
