# Long Hold Dividend V4 Refactor Report

## 目标变化

项目不再以行业轮动跑赢中证全指为主目标。新主线只研究可以多年持有、分红稳定、增长稳定且波动相对有限的股票和ETF，并解决两个独立问题：低估企稳时分批建立核心仓；核心仓建立后用受限T仓降低成本。

## 已替换的核心设计

| 旧设计 | V4设计 |
|---|---|
| 风格/行业指数轮动 | 银行、保险、公用事业及高股息ETF耐久性筛选 |
| 排名后强制配置 | 不满足条件时保持现金 |
| 全局专家混合 | 质量、估值、陷阱、时机、T、组合按顺序硬交接 |
| 单次目标仓位 | 30%、60%、100%三批建仓 |
| 价格反转专家 | 大幅回撤、低估、停止加速下跌同时成立 |
| 做T概念描述 | 核心/T分账、成本门槛、T+1、次日开盘执行 |
| 重叠标签绩效 | 事件驱动逐日持仓回测 |
