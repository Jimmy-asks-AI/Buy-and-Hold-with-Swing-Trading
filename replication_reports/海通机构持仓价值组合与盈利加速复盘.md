# 复盘：机构持仓、价值组合与盈利加速

## 复盘目标

将四篇海通选股因子研报中的可复用方法转化为可验证研究模块，而不是直接接受报告收益率。

## 可复现实验设计

### 实验1：基金重仓超配因子

数据：

- 主动股票型基金、偏股/灵活配置混合基金十大重仓。
- 基金过去 1 年收益率，用于前50%/后50%分组。
- 全A或目标指数成分权重。
- 股票月收益、风格因子、行业因子。

步骤：

1. 按基金季报披露可得日期建立持仓快照。
2. 聚合持仓市值为 `fund_pool_weight`。
3. 与基准权重比较，构造 `fund_overweight_flag` 与 `overweight_gap`。
4. 月频测试多头、空头、多空收益。
5. 在横截面回归中加入该虚拟变量，记录显著比例、正负显著比例。
6. 用过去 12 个月因子溢价均值做择时门控，比较直接加入、负向置零、允许反向三种方法。

必须验证：

- 披露滞后处理。
- 按基金业绩分组时的小样本边界。
- 溢价方向延续比例。
- 与市值、估值、行业、动量的相关性。

### 实验2：深度价值与低估值组合

数据：

- 资产负债表、股本、价格、分红、估值、行业、市值、换手率、SUE。

步骤：

1. 计算 NCAV1/NCAV2，筛选 `price < NCAV`。
2. 季报披露后等待 3 个月买入，持有 1 年，统计股票数量分布。
3. 在连续现金分红股票池中复合 PB/PE/PCF/股息率，构建 Top20/50/100。
4. 对沪深300、中证500、中证1000分别构造纯价值组合，控制市值、换手率、SUE和成分股权重。
5. 拆分行业中性与行业放开，记录收益、波动、回撤和1月效应。

必须验证：

- NCAV 筛选方向使用 `price < NCAV`。
- 深度价值容量是否足够。
- 价值收益来自个股选择还是行业暴露。
- 2019-2020成长风格期的回撤表现。

### 实验3：有基本面支撑的价值组合

PB-盈利：

1. 盈利得分 = ROE + SUE。
2. 估值得分 = 低PB。
3. 取盈利最高20%与低PB最高20%交集。
4. 加入低换手率过滤。
5. 加入位序估值过滤：长期相对估值不能低于10%，短期相对估值不能高于90%。

估值-增长：

1. 增长得分 = 一致预期2年CAGR + 预期净利润调整 + SUE。
2. 估值得分 = PB + PE + 股息率。
3. 取增长多头与估值多头交集，再做低关注度和位序估值过滤。

必须验证：

- 分析师预期数据的可得日期。
- 交集股票数量、行业集中度、个股最大权重。
- 等权、价值加权、市值加权对结果是否敏感。
- 参数从20%改为10%/30%是否稳定。

### 实验4：盈利加速与高增长组合

数据：

- 单季度 EPS、价格、SUE、预期净利润调整、研发资本化指标、PB_INT、行业、市值。

步骤：

1. 分别计算 EAA/EAP/EAV。
2. 对比多头、多空、IC、RankIC。
3. 对常见风格、基本面、预期类、价量类和行业做正交，验证 EAV 的增量信息。
4. 按市值分层和 SUE 分层测试。
5. 按 6 种盈利加速模式测试组合收益。
6. 构建高增长 Top50，并加入高频因子做增厚实验。

必须验证：

- 季报披露日期，避免用季度末提前获得 EPS。
- EAV 分母使用近 8 季 EPS 标准差，极小分母要过滤。
- 高增长组合的风格归因，特别是小市值、高估值、SUE暴露。
- 单边千三之外，要测试更高交易成本和涨跌停不可成交。

## 复习纠错记录

- 报告文字中关于 NCAV 与股价关系存在容易误读处。深度价值定义应采用 `price < NCAV`，代码按该定义实现。
- 初版代码用百分位排名筛基金前/后50%，在只有两只基金时会把前50%扩成全部基金。已改为每期按 `ceil(n * q)` 取前/后若干只。
- 基金重仓超配的截面显著性不能推出稳定正向 alpha，必须做滚动溢价方向判断。
- 价值+增长报告收益很高，不能忽视分析师预期库、幸存者偏差、停牌涨跌停、换手成本和小组合容量。
