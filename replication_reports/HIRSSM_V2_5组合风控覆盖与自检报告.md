# HIRSSM V2.5 组合风控覆盖与自检报告

日期：2026-05-24

## 本轮目标

V2.5 不新增 alpha 专家，不扩大参数网格，而是在 V2.4 稳定嵌套选择目标权重之上叠加固定组合风控层：

- 状态条件化目标波动。
- 状态条件化总风险暴露上限和最低现金。
- 市场回撤刹车。
- 组合滚动回撤刹车。
- 拥挤资产降权。
- 低质量候选现金替代。

目标不是提高全样本收益，而是检验固定、可解释风控是否能显著降低回撤，并保持成本后收益不坍塌。

## 新增/修改文件

- `strategy_lab/hirssm_v2_5_portfolio_risk_overlay.py`
- `configs/hirssm_v2_default.json`
- `outputs/hirssm_v2_5_portfolio_risk_overlay/`

## 自检纠错过程

第一版 V2.5 工程 smoke 通过，但自检失败：

- 10bps 年化收益从 V2.4 的 7.73% 降到 4.50%。
- 平均现金升到 45.25%。
- 主要原因是动态回撤刹车使用成立以来高水位回撤，导致一次大跌后长期处于降风险状态，变成“永久高现金”。

修复：

- 将组合动态回撤刹车改为近 252 个交易日滚动回撤。
- 将状态目标波动和最低现金参数调回更温和的固定政策。
- 重新跑完整回测和自检，而不是手工改输出。

## 刹车诊断

10bps 动态回撤刹车：

- 未触发：192 次。
- 软刹车：30 次。
- 硬刹车：10 次。
- 触发率：17.24%。
- 平均计划总风险暴露：69.98%。
- 平均执行总风险暴露：68.92%。

滚动回撤修复后，刹车不再长期压制仓位。

## 自检结果

通过：

- Python 语法编译。
- JSON 配置解析。
- 目标权重 smoke test。
- 5/10/20/30bps 输出齐全。
- 10bps 最大回撤不劣于 V2.4。
- 10bps 年化收益相对 V2.4 回落未超过 2pct 阈值。
- 10bps 平均现金低于 40% 阈值。
- 20bps 年化收益为正。
- 无新增 alpha 或新增选择网格。
- 必要报告输出齐全。

## 结论

V2.5 通过自检，但不应直接替代 V2.4 成为唯一默认模型。

更合理的定位：

- V2.4 是收益更高的研究基线。
- V2.5 是风险控制版候选，适合“稳健可投/低回撤优先”的分支。
- 由于 V2.4 的 PBO 问题仍未彻底解决，V2.5 也继承该底层 alpha 稳定性风险；V2.5 只能说明固定风控层有效降低回撤，不能证明底层 alpha 已经生产级稳健。

下一步 V2.6 应做分情景归因：确认 V2.5 的收益损失主要发生在哪些市场状态、哪些年份、哪些资产 sleeve，并尝试把风控从全组合缩放升级为“状态 + sleeve + 风险来源”的局部降权。
