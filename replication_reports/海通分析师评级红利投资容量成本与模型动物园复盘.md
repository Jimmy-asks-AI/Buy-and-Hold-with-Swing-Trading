# 海通分析师评级、红利投资、容量成本与模型动物园复盘

## 目标

本轮把四份海通选股因子报告转化为可复用研究单元：行为因子、红利风格择时、红利+组合、交易容量校正、深度学习模型集成。

## 可复现定义

### 买入评级

1. 从分析师报告库读取报告日期、股票、评级、报告类型、分析师。
2. 仅保留买入/增持/推荐类评级。
3. 仅保留点评和深度报告。
4. 按月聚合到股票层面，补全缺失月份。
5. 生成：
   - `buy_rating`
   - `new_buy_rating`
   - `continuous_buy_rating`
   - `buy_report_count`
   - `analyst_count`
6. 与 SUE 合并，排除 SUE 底部 1/3，形成 `supported_new_buy_rating`。

### 红利+

红利类策略必须先处理分红披露和可得性。基础股票池至少应排除 ST、新股、低成交额、低市值和支付率异常股票。

- 红利+成长：股息率 + SUE 形成基础池，然后高股息优先，再用 SUE 收敛到 50 只。
- 红利+低波：基本面质量形成基础池，高股息优先，再用低波和低换手收敛到 50 只。
- 分红潜力：连续分红池中使用 ROE、SUE、盈利质量、分析师观点、低换手选股。

### 容量与成本

1. 根据调仓频率选择成交额口径：月频用全天，周频用开盘后半小时。
2. 使用 `5% * 成交额 >= 0.1% * 组合规模` 初筛。
3. 分规模计算可交易股票比例与因子 Rank IC。
4. 用盘口委托额、价格波动、买卖价差和大单冲击构建总成本。
5. 总成本超过 1% 的股票剔除后重新计算 IC。

### 模型动物园

1. 固定输入特征、标签和训练设定。
2. 分别训练 RNN、卷积、Transformer、线性模型。
3. 用 T0 收盘 Rank IC、T1 均价 Rank IC、Top10%/Top100 组合费前费后超额、换手、自相关评价。
4. 对模型分数按日期 z-score 后等权集成。
5. 诊断模型与低流动性、反转、低波动因子的暴露漂移。

## 复习纠错

- `concept_fix`: 买入评级不能用全量报告粗糙构造，应区分点评/深度报告、新增/连续评级和基本面支持。
- `concept_fix`: 红利风格择时是相对风格择时，不是单独的市场仓位择时。社融下降尤其只说明相对成长更强。
- `method_fix`: 红利+策略要明确红利属性强弱，分红潜力组合不是纯高股息组合。
- `method_fix`: 因子容量测试必须以真实交易时段成交额为基础；周频策略不能套用全天成交额。
- `method_fix`: 深度学习模型动物园的目标不是追逐最复杂结构，而是建立基线、识别暴露差异并做稳健集成。
- `code_fix`: 把上述规则固化为四个模块，并在烟测中检查时点、缺失期、权重、容量和模型相关性。
