# NLP入门与基金池标签框架复盘

## 复盘对象

- NLP 入门科普 PDF。
- 华安基金分类标签框架。
- 浙商权益基金池体系与研究框架。

## NLP复盘

NLP 入门材料提供了文本研究的基础语言：NLU、NLG、分词、词性标注、NER、情感分析。迁移到金融场景后，不能只关注模型效果，还要重点处理发布时间、实体映射和交易可得性。

文本因子的最小可靠流程是：

```text
文本 -> 清洗 -> 实体识别 -> 情绪/事件打分 -> 可得交易日对齐 -> 资产日聚合 -> 滞后滚动特征 -> 因子验证
```

如果缺少交易日对齐，文本因子很容易把收盘后新闻、公告后财报或研报发布日期错误用于过去交易。

## 基金分类复盘

华安报告的关键贡献是指出基金评价必须先找对“锚”。WIND 合同分类容易把实际仓位、选股范围和投资策略差异很大的基金放在一起，导致业绩评价不可比。

更合理的体系是：

- 先按大类资产划分风险收益属性。
- 再按管理方式和选股范围细分。
- 对主题基金使用合同、基准、持仓三重证据。
- 对不同类别建立不同评价指标和标签。

## 基金池复盘

浙商报告更偏买方工作流：基础池、优选池、观察池、投资池。这个框架强调有限研究精力要聚焦到可深度研究对象上，并且基金入池后还要持续跟踪业绩、持仓、基金经理动态和重大变化。

主动权益基金研究不能只看净值曲线，应把定量和定性结合：

- 净值表现用于观察结果。
- 持仓归因用于解释来源。
- 基金经理调研用于理解框架。
- 历史操作复盘用于检验知行合一。
- 持续跟踪用于判断能力是否变化。

## 纠错

- 文本模型好不等于文本因子可交易；时间戳和实体映射是前置条件。
- 基金历史业绩好不等于基金经理能力稳定；要看同类基准、风格、持仓、规模和策略变化。
- 主动主题基金跑赢被动基准不代表所有主题都适合主动管理；主题内标的广度和持股异质度会影响主动管理空间。
