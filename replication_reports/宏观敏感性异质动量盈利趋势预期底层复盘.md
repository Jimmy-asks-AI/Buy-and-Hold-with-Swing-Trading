# 宏观敏感性、异质动量、盈利趋势与预期底层复盘

## 复盘对象

- 宏观指标选股：MacroBeta 与 T 值筛选。
- 异质动量：共同因子剥离后的残差动量。
- 盈利趋势：Gross Profitability 的滚动 OLS 趋势。
- 一致预期底层数据：不同财年选择法。

## 可复现假设

1. 宏观因子有效性来自“显著敏感股票 + 可得宏观方向”，不是原始 beta 排序。
2. IMom 有效性来自共同风险剥离后的个股特质趋势。
3. GP 同比趋势比单点 GP 和环比趋势更能捕捉盈利改善。
4. 一致预期因子先处理底层财年，才能再做时间序列标准化。

## 反证清单

- MacroBeta 若不经过 t 值筛选，可能只是噪声排序。
- IMom 若无法战胜 raw momentum 或低波动、低换手控制变量，不能单独入库。
- GP 趋势若与当前 GP 高度重合且无增量 IC，应作为质量因子的变体而非新增因子。
- 锁定财年法若覆盖率过低，可能牺牲组合可交易性。
- 所有报告结论都必须在真实 A 股可得日数据上复验，研报中的收益数字只作为复现目标，不作为直接承诺。

## 修正记录

- 将宏观敏感性从 raw beta 排序修正为 beta + t-stat + macro forecast。
- 将动量从 raw return 修正为共同因子残差动量。
- 将盈利改善从同比增速修正为可比季度 OLS 趋势。
- 将一致预期调整从单一字段读取修正为底层财年选择后再标准化。
