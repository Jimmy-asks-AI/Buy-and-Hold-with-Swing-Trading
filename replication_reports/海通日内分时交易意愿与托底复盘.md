# 海通日内分时交易意愿与托底复盘

Date: 2026-05-22
Batch: Intraday Trade, Commission Intention, Support Behavior

## 复盘对象

- 海通证券《选股因子系列研究（四十六）：日内分时成交中的玄机》
- 海通证券《选股因子系列研究（四十七）：捕捉投资者的交易意愿》
- 海通证券《选股因子系列研究（四十九）：当下跌遇到托底》

## 原始研究设置

共同思路：

- 股票池剔除 ST、停牌、涨跌停、次新、临近退市等不可交易样本。
- 高频数据聚合为每日或月度截面因子。
- 对行业、市值、中盘、换手、反转、波动、估值、盈利、成长等常见暴露做中性化或回归检验。
- 使用 IC、Rank IC、ICIR、分组收益、多空收益和 Fama-MacBeth 回归评价。

《日内分时成交中的玄机》的关键设置：

- 回测区间：2010.01-2019.03。
- 成交价：次日开盘后半小时 VWAP。
- 交易成本：双边千分之三。
- 剔除集合竞价；若日内超过 25% 时间价格不动，剔除该股票日。

## 可复现实验拆解

### 实验 1：分钟成交金额因子

输入字段：

- `date`, `asset`, `minute_time`
- `amount`
- `trade_count`
- `minute_return`

复现步骤：

1. 计算每分钟平均单笔成交金额。
2. 分别聚合上涨分钟、下跌分钟的平均单笔成交金额。
3. 用全日平均单笔成交金额标准化，得到流入/流出金额占比。
4. 用过去 20 个交易日均值生成月度因子。
5. 做截面 winsorize、标准化、中性化。
6. 测试未来 1 个月收益，另补充 1/3/5/10/20 日周期。

对应代码：

- `strategy_lab/order_flow_factors.py::minute_trade_size_features`
- `strategy_lab/order_flow_factors.py::rolling_mean_by_asset`

### 实验 2：大单分钟资金流

输入字段同实验 1。

复现步骤：

1. 每个股票日按 `amount / trade_count` 排序。
2. 取前 10%、20%、30% 的分钟作为大单成交分钟。
3. 计算大单分钟净流入率和大单驱动收益。
4. 对 N 做参数敏感性。
5. 重点验证方向：报告中大单净流入率和大单驱动涨幅为负向预测。

对应代码：

- `strategy_lab/order_flow_factors.py::minute_trade_size_features`

### 实验 3：盘口净委买变化率

输入字段：

- `bid_volume_1...bid_volume_k`
- `ask_volume_1...ask_volume_k`
- `float_shares`
- `time`

复现步骤：

1. 计算前 k 档委买量变化和委卖量变化。
2. 得到 `net_bid_change_rate = (delta_bid - delta_ask) / float_shares`。
3. 按全天、开盘后、盘中、收盘前聚合均值、波动率、偏度。
4. 分别测试前 1 档、前 5 档、前 10 档和全部档位。
5. 检查前 1 档是否显著优于深档位。

对应代码：

- `strategy_lab/order_flow_factors.py::add_net_bid_change_rate`
- `strategy_lab/order_flow_factors.py::net_bid_change_rate_features`

生产级修正：

- 简单按固定档位做差只是模板，真实复现需处理最优价变化导致的档位迁移。
- 涨跌停、停牌、无成交快照需要独立剔除或标记。

### 实验 4：委托成交相关性与托底形态

输入字段：

- 高频收益率。
- 净委买变化率。
- 时段标签。

复现步骤：

1. 对每个股票日计算 `corr(minute_return, net_bid_change_rate)`。
2. 用过去窗口均值生成因子。
3. 测试全日、开盘后、盘中、收盘前。
4. 对低频因子和已有高频因子做中性化。
5. 用净委买变化率均值和委托成交相关性做双重分组，识别托底形态。

形态解释：

- `price_down_net_bid_up_support`: 高净委买、低相关，报告中后续收益最强。
- `price_down_net_bid_down_weak_support`: 低净委买、高相关，报告中后续收益最弱。
- `price_up_net_bid_down`: 有一定超额收益，但弱于托底形态。
- `price_up_net_bid_up`: 直觉上偏强，但报告结果偏弱。

对应代码：

- `strategy_lab/order_flow_factors.py::order_trade_correlation_features`
- `strategy_lab/order_flow_factors.py::classify_order_trade_shapes`

## 复盘结论

1. 平均单笔流出金额占比是较稳的分钟成交因子，但更像短周期反转/冲击补偿。
2. 大单净流入和大单驱动上涨不能机械解释为利好，报告实证方向为负。
3. 盘口挂单数据补充了“未成交意愿”，但撤单和档位迁移会带来噪声。
4. 委托成交相关性是这批材料中最有结构的信息：低相关不是简单反转，而是可能反映“下跌中买盘托底”。
5. 对长线高分红低估值框架，最佳用法是做个股微观择优、风险过滤和做 T 状态识别，不应直接替代估值择时。
