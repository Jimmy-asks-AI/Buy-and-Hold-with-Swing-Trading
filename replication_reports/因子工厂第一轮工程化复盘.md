## 工程能力

现在可以执行：

```text
python -X utf8 strategy_lab/factor_factory_runner.py \
  --panel-csv panel.csv \
  --registry-csv factor_registry.csv \
  --config-json configs/factor_factory_default.json \
  --output-dir outputs/factor_factory_run_xxx
```

也可以运行合成数据 demo：

```text
python -X utf8 strategy_lab/factor_factory_runner.py \
  --synthetic-demo \
  --output-dir outputs/factor_factory_demo
```

实验完成后写入账本：

```text
python -X utf8 strategy_lab/factor_factory_ledger.py \
  --output-dir outputs/factor_factory_demo \
  --ledger logs/factor_factory_experiment_ledger.csv \
  --experiment-id demo_001_synthetic_factor_factory \
  --hypothesis "synthetic pipeline should recover planted factor families and produce auditable outputs"
```

## 纠错

1. `Series.reset_index(names=...)` 在当前 pandas 环境不可用，已改为 `reset_index().rename(...)`。
2. 空 `availability_audit.csv` 会触发 `EmptyDataError`，已改为读取空表。
3. 合成 demo 的高绩效不应解读为模型有效，只说明 planted signal 能被工程链路识别。
