# Formal Data Intake Example

本目录只提供字段示例，不包含真实行情、授权数据或可运行的正式输入。

- `point_in_time_usage.example.csv`：历史决策与源数据行绑定。
- `candidate_registry.example.csv`：候选只登记参数和训练信息，不能自报验证成绩。
- `benchmark_returns.example.csv`：中证全指全收益日收益格式。
- `intake_config.example.json`：正式入库配置骨架，10 类数据应各有一项。

使用前阅读 `data_catalog/long_hold_v4_formal_evaluation_data_contract_v1.md`。示例中的路径、日期、供应商和修订号都必须替换为真实、获授权且可审计的内容。
