## 课程目标

学完第一阶段后，你应该能清楚区分：

- 量化研究、量化交易、程序化交易不是一回事
- 数据、指标、因子、信号、策略、组合、回测、风控分别是什么
- 为什么历史回测不能直接等于未来收益
- 为什么因子有效不等于策略能赚钱
- 为什么收益率必须和风险、回撤、换手、成本一起看
- 为什么很多看似有效的策略其实可能是过拟合或未来函数

这门课程先解决“听得懂、问得准、能判断”的问题。代码和完整研究流程放在第二阶段逐步展开。

## 学习原则

### 先概念，后代码

代码只是表达研究想法的工具。我们先讲清楚每个概念在研究中扮演什么角色，再去看项目里的 Python 示例。

### 先简单，后复杂

先学日频行情、简单技术指标、基础因子、简单回测。暂时不急着进入高频、机器学习、强化学习、复杂组合优化。

### 先会判断，后追求收益

初学阶段最重要的不是找到赚钱策略，而是能判断一个研究是否靠谱。

## 第一阶段：概念入门

### 第 1 讲：什么是量化研究

核心问题：

> 量化研究到底在研究什么？

你需要理解：

- 量化研究是用数据和规则验证投资假设
- 量化交易是把研究结果转成交易执行
- 程序化交易是用程序自动执行交易规则
- 研究不等于实盘，回测不等于赚钱

关键概念：

- 投资假设
- 可验证问题
- 规则化
- 可复现
- 样本内
- 样本外

例子：

```text
模糊说法：
低估值股票长期更好。

量化研究说法：
在 A 股全市场中，每月底按 PB 从低到高排序，买入最低 20% 股票，等权持有一个月，比较其未来收益是否显著高于全市场平均。
```

本讲产出：

你能把一个模糊投资想法改写成一个可验证问题。

### 第 2 讲：研究对象是什么

核心问题：

> 我们到底在研究什么资产？

你需要理解：

- 股票、ETF、指数、基金、期货、债券是不同资产
- 不同资产的数据、交易规则、风险来源都不同
- A 股有涨跌停、停牌、退市、T+1 等特殊约束

关键概念：

- 股票池
- 指数成分
- ETF
- 行业
- 交易日
- 流动性
- 可交易性

例子：

```text
研究沪深 300 成分股，和研究全 A 股，不是同一个问题。
研究 ETF 轮动，和研究个股多因子，也不是同一个问题。
```

本讲产出：

你能说清楚一个研究的对象范围：研究哪些资产，不研究哪些资产。

### 第 3 讲：数据是什么

核心问题：

> 量化研究需要哪些数据？

你需要理解：

- 行情数据回答“价格和成交发生了什么”
- 基本面数据回答“公司经营情况如何”
- 估值数据回答“当前价格相对基本面贵不贵”
- 预期数据回答“市场对未来怎么看”
- 另类数据回答“传统财务和行情以外的信息”

关键概念：

- 开盘价
- 最高价
- 最低价
- 收盘价
- 成交量
- 成交额
- 前复权
- 后复权
- 停牌
- 缺失值

项目资料：

- `资料/数据源与另类数据.md`

本讲产出：

你能解释为什么量化研究首先要关心数据口径。

### 第 4 讲：收益和风险

核心问题：

> 策略赚了多少，承担了多大风险？

你需要理解：

- 收益率不是绝对金额，而是相对本金的变化
- 年化收益方便比较不同时间长度的策略
- 波动率衡量收益的不稳定程度
- 最大回撤衡量从高点到低点最痛的一段亏损
- 夏普比率尝试衡量单位风险带来的收益

关键概念：

- 日收益率
- 累计收益
- 年化收益
- 波动率
- 最大回撤
- 夏普比率
- 胜率
- 盈亏比

例子：

```text
策略 A 年化收益 20%，最大回撤 50%。
策略 B 年化收益 12%，最大回撤 8%。

不能只因为 A 收益高就认为 A 更好。
```

本讲产出：

你能说明为什么“收益高”不等于“策略好”。

### 第 5 讲：指标、因子、信号、策略

核心问题：

> 这些词到底有什么区别？

这是量化入门最重要的一讲。

你需要理解：

- 指标：从原始数据加工出来的数值
- 因子：可能解释或预测未来收益的变量
- 信号：把指标或因子转成买卖、持有、回避等动作提示
- 策略：完整的交易规则，包括买什么、什么时候买、买多少、什么时候卖

例子：

```text
KDJ 的 K 值、D 值是指标。
K 上穿 D 是信号。
规定金叉买入、死叉卖出、每次满仓一只股票，才接近策略。

PB 是估值指标。
低 PB 可能是价值因子。
每月买入 PB 最低的 50 只股票，是策略规则的一部分。
```

项目资料：

- `资料/技术指标回测代码/technical.py`
- `资料/因子挖掘.md`

本讲产出：

你能区分指标、因子、信号、策略，不再混用这些词。

### 第 6 讲：回测是什么

核心问题：

> 回测是在做什么，又最容易错在哪里？

你需要理解：

- 回测是用历史数据模拟当时按规则交易会发生什么
- 回测必须尊重当时能看到的信息
- 回测必须考虑交易成本和交易限制
- 回测结果只是证据，不是保证

关键概念：

- 未来函数
- 滑点
- 手续费
- 印花税
- 调仓
- 持仓
- 基准
- 样本内
- 样本外
- 过拟合

项目资料：

- `资料/回测.md`
- `资料/技术指标回测代码/main.py`

本讲产出：

你能指出一个回测最应该检查的几个风险点。

### 第 7 讲：基准和超额收益

核心问题：

> 策略赚钱，是因为策略强，还是因为市场涨了？

你需要理解：

- 基准是用来比较策略表现的参照物
- 绝对收益是策略自身赚了多少
- 超额收益是策略相对基准多赚了多少
- 做股票策略必须关心市场 Beta

关键概念：

- 基准指数
- 绝对收益
- 相对收益
- 超额收益
- Alpha
- Beta
- 跟踪误差
- 信息比率

例子：

```text
策略一年赚 15%，但同期沪深 300 涨 25%。
这不是一个优秀结果，因为它跑输了基准。
```

本讲产出：

你能解释为什么策略评价必须有基准。

### 第 8 讲：因子研究入门

核心问题：

> 什么样的变量可以被称为因子？

你需要理解：

- 因子是对未来收益可能有解释力的变量
- 因子通常用于横截面排序
- 因子需要有经济逻辑和统计证据
- 因子有效不代表一定能交易

常见因子类型：

- 价值：PE、PB、股息率
- 质量：ROE、毛利率、现金流质量
- 成长：收入增长、利润增长
- 动量：过去一段时间涨幅
- 反转：短期跌幅或涨幅反向修复
- 波动：历史波动率
- 流动性：换手率、成交额

项目资料：

- `资料/因子挖掘.md`
- `资料/卖方金工研报/海通报告/`

本讲产出：

你能为一个因子说清楚：它是什么、怎么算、为什么可能有效。

### 第 9 讲：因子怎么评价

核心问题：

> 怎样判断一个因子有没有用？

你需要理解：

- 单次收益不能证明因子有效
- 因子评价通常看相关性、分组收益和稳定性
- 因子要看长期、多阶段、多市场环境表现

关键概念：

- IC
- Rank IC
- ICIR
- 分组收益
- 多空组合
- 单调性
- 因子胜率
- 稳健性

例子：

```text
如果按因子从低到高分成 10 组，
第 10 组长期收益最高，第 1 组长期收益最低，
中间组大致单调，
这比只看最高组某一年收益更有说服力。
```

本讲产出：

你能看懂一份基础因子评价表。

### 第 10 讲：组合是什么

核心问题：

> 选出股票之后，为什么还需要组合？

你需要理解：

- 策略最终持有的是组合，不是一个孤立因子
- 同样的选股结果，不同权重会产生不同收益和风险
- 组合管理要处理集中度、行业、风格和流动性

关键概念：

- 等权
- 市值加权
- 仓位
- 个股权重
- 行业权重
- 调仓频率
- 换手率
- 集中度

项目资料：

- `资料/投资组合优化与风控.md`

本讲产出：

你能说明为什么“选什么”和“买多少”是两个问题。

### 第 11 讲：风险控制

核心问题：

> 研究中要防哪些风险？

你需要理解：

- 风险不只是亏钱，还包括亏钱的路径和原因
- 研究风险来自数据、模型、交易、市场环境和人的误判
- 风控不是让策略永远不亏，而是让亏损可理解、可承受、可复盘

关键概念：

- 最大回撤
- 波动率
- 行业暴露
- 风格暴露
- 流动性风险
- 拥挤交易
- 黑天鹅
- 参数过拟合

本讲产出：

你能列出一个策略在实盘前需要检查的风险清单。

### 第 12 讲：量化研究报告

核心问题：

> 一个研究怎么留下来，方便以后复盘？

你需要理解：

- 研究必须能被复现
- 每次研究都要记录假设、数据、方法、结果和缺陷
- 好的研究报告不是展示漂亮收益，而是说明证据是否支持假设

报告结构：

```text
1. 研究问题
2. 投资逻辑
3. 数据范围
4. 指标、因子或策略定义
5. 回测设置
6. 结果分析
7. 风险与缺陷
8. 结论
9. 下一步
```

本讲产出：

你能按固定模板写一页量化研究笔记。

完成第一阶段后，我们用项目里的 KDJ 示例做一次完整拆解。

研究问题：

```text
KDJ 金叉后，未来 1、5、10、20 日收益是否更好？
```

对应文件：

- `资料/技术指标回测代码/technical.py`
- `资料/技术指标回测代码/main.py`
- `资料/技术指标回测代码/技术指标编写说明.txt`

我们会按这个顺序学习：

```text
数据字段 -> KDJ 指标 -> 金叉死叉信号 -> 未来 N 日收益 -> 统计结果 -> 结论和问题
```

这一阶段你需要理解的重点：

- 这只是信号验证，不是完整交易系统
- 未来收益只能用于评价，不能用于生成当日信号
- 统计结果要看样本数量、均值、分布和稳定性
- 如果结果有效，还要继续做严格回测

## 第三阶段：因子与策略入门

在理解 KDJ 示例后，我们进入更标准的量化研究。

推荐学习顺序：

```text
简单技术指标 -> 简单价量因子 -> 估值因子 -> 质量因子 -> 多因子组合 -> 回测和风控
```

优先研究这些简单因子：

- 20 日动量
- 20 日反转
- 20 日波动率
- 20 日平均成交额
- PE
- PB
- ROE
- 股息率

暂时不优先研究：

- 高频因子
- 深度学习预测股价
- 强化学习交易
- 复杂遗传规划挖因子
- 实盘自动交易

原因是基础概念没稳之前，复杂模型只会放大错误。

## 必懂概念速查表

| 概念 | 简单解释 |
| --- | --- |
| 数据 | 研究的原材料 |
| 指标 | 从数据加工出的数值 |
| 因子 | 可能解释或预测收益的变量 |
| 信号 | 因子或指标触发的动作提示 |
| 策略 | 完整交易规则 |
| 回测 | 用历史数据模拟策略表现 |
| 基准 | 策略比较对象 |
| 超额收益 | 策略相对基准多赚的部分 |
| 回撤 | 从高点到低点的亏损幅度 |
| 波动率 | 收益变化的不稳定程度 |
| 夏普比率 | 单位风险带来的收益 |
| IC | 因子和未来收益的相关性 |
| 分组收益 | 按因子高低分组后的收益表现 |
| 换手率 | 持仓变化的频繁程度 |
| 交易成本 | 买卖产生的费用和冲击 |
| 未来函数 | 使用了当时不可能知道的信息 |
| 过拟合 | 策略只适应历史样本，不适应未来 |
| 风控 | 控制风险暴露和亏损路径 |
