# 高频空头效应与多头失效

## Purpose

识别“整体有效但多头端失效”的高频因子，并把空头端信息以更稳健的方式纳入组合。

## Diagnostics

```text
long_short_return = long_group_return - short_group_return
long_excess = long_group_return - universe_return
short_excess = universe_return - short_group_return
long_contribution_ratio = long_excess / long_short_return
```

若出现以下特征，应标记为多头失效候选：

- 整体 IC 显著，但多头组 IC 弱或为负。
- 多空收益主要来自空头端。
- 加入收益预测模型后组合收益没有提升。
- 加权 IC 明显低于原始 IC。

## Weighted IC

```text
weighted_ic = corr_w(aligned_factor, fwd_return)
```

用法：

- 先把因子方向统一为“越高越好”。
- 按因子值分组。
- 提高多头组权重，例如多头组 50%，其余组均分 50%。
- 若加权 IC 显著下降，说明该因子对多头端贡献不足。

## Short-Leg Usage

可选方法：

- `pre_exclusion`: 优化前剔除或惩罚短端股票。
- `short_dummy_factor`: 短端股票设为 1，其余设为 0，加入收益预测模型。
- `short_exposure_constraint`: 对短端组合暴露加约束。
- `post_exclusion`: 优化后剔除短端股票并重分配权重。
- `delay_buy`: 对计划买入但短端风险高的股票延迟买入。

## Multi-Factor Short Portfolio

```text
zscore_composite = sum(weight_i * zscore(sign_i * factor_i))
short_flag = zscore_composite <= quantile(zscore_composite, q)
```

或：

```text
short_membership_count = sum(single_factor_short_flag_i)
short_flag = short_membership_count >= M
```

经验规则：

- 剔除比例通常控制在 4%-10%。
- 沪深300中，构建复合因子时可只在成分股外样本内计算，以避免扭曲基准。
- 中证500中，组合复合事后剔除通常比因子复合、事前剔除更稳。

## Nonlinear Repair

若多头失效来自非线性关系：

- 加入 `factor^2`、`factor^4`。
- 使用 RBF 升维，把因子值分段后再线性拟合。

风险：

- 维度增加降低参数稳定性。
- 月度回归样本少，容易过拟合。
- RBF 中心点数量和训练窗口敏感。

## Failure Modes

- 把空头端有效因子误当成多头信号。
- 空头阈值过高导致空头效应被稀释。
- 剔除比例过高破坏原模型稳定性。
- 事后剔除不控制行业和风格偏离。
- 多因子回归复合未处理因子相关性，导致方向翻转。
- 非线性升维样本外失效。
