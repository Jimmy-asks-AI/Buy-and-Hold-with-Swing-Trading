# 因子卡：机构持仓、价值支撑与盈利加速

## 适用范围

- A股基本面选股、指数增强、行业内选股、高股息/价值/成长混合框架。
- 频率以月频、季报披露频率为主；基金持仓、财报、分析师预期数据必须做可得日期处理。

## 因子定义

| 因子 | 方向 | 主要数据 | 更新频率 | 用法 |
|---|---:|---|---|---|
| 基金重仓超配 | 不固定 | 基金季报十大重仓、基准权重 | 季度披露后 | 先做溢价方向择时，再进入 alpha 或过滤层 |
| NCAV 深度价值 | 股价低于 NCAV 更好 | 资产负债表、股本、价格 | 季报 | 小容量深度价值观察池，持有期偏长 |
| 低估值复合 | 更低 PB/PE/PCF、更高股息率 | 财务、分红、价格 | 月/季 | 价值暴露，需控制市值、SUE、换手率、行业 |
| PB-盈利 | 低PB + 高ROE/SUE | 财务、SUE、价格 | 月/季 | 有基本面支撑的价值组合 |
| 估值-增长 | 低估值 + 高预期增长/SUE/修正 | 一致预期、财务、估值 | 月 | 价值与成长交集组合 |
| EAA | EPS增速变化/去年同期EPS绝对值 | EPS | 季报 | 效果弱于 EAP/EAV |
| EAP | EPS增速变化/上季末价格 | EPS、价格 | 季报 | 盈利加速备选 |
| EAV | EPS增速变化/近8季EPS标准差 | EPS | 季报 | 盈利加速主因子 |
| 高增长复合 | EAV+SUE+预期修正+研发资本+PB_INT | 财务、预期、无形资产估值 | 月/季 | 成长组合核心打分 |

## 关键构造

基金重仓超配：

1. 取可投资基金池的十大重仓股并集。
2. 按报告期聚合持仓市值，得到 `fund_pool_weight`。
3. 与全A或目标指数权重比较：`overweight_gap = fund_pool_weight - benchmark_weight`。
4. `overweight_gap > 0` 得到超配虚拟变量。
5. 使用披露可得日期，不能使用报告期末日期。
6. 对因子历史溢价做滚动均值，若预估溢价为负，可将因子权重置零。

深度价值：

- `NCAV1 = [现金 + 0.75 * 应收账款 + 0.5 * 存货 - (总负债 + 优先股)] / 流通股本`
- `NCAV2 = [0.75 * 流动资产 - (总负债 + 优先股)] / 总股本`
- 标准定义为 `price < NCAV`。这类股票数量少、等待期长，应独立管理容量。

有支撑的价值组合：

1. 计算低估值得分，方向统一为“越高越便宜”。
2. 计算盈利/增长支撑得分，如 ROE、SUE、预期净利润调整、预期 CAGR。
3. 取价值多头与支撑多头交集。
4. 可叠加低换手率/低关注度过滤。
5. 用相对估值位序过滤价值陷阱：剔除长期极端低位和短期估值快速攀升。

盈利加速：

1. `growth_t = (EPS_t - EPS_{t-4}) / denominator_t`
2. `acceleration_t = growth_t - growth_{t-1}`
3. 首选 EAV：`denominator_t = std(EPS_{t-7}, ..., EPS_t)`
4. 入模前需要与 SUE、盈利水平、行业、风格做正交或归因诊断。

## 失效条件

- 基金重仓超配在机构抱团瓦解或风格反转时可能持续负溢价。
- 低估值组合在成长风格强势期可能跑输，行业放开时回撤更大。
- 深度价值可能长期不修复，且个股数量太少导致组合不稳定。
- 盈利加速依赖财报质量和披露时点，季报重述或业绩预告冲击会造成噪声。
- 高增长组合通常暴露于小市值、高估值、高SUE或TMT行业，需要风格归因后使用。

## 研究质量门槛

- 必须按可得日期对齐基金持仓、财报、分析师预期。
- 必须分别报告原始因子、风格中性后因子、行业中性后因子。
- 必须检查多头收益、多空收益、IC、RankIC、换手、容量、行业集中、风格暴露。
- 对方向不稳定的因子，必须报告滚动溢价方向延续比例。
- 小股票数策略必须报告持仓数量分布和最大单票权重。
