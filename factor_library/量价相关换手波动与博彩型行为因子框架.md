# 量价相关、换手波动与博彩型行为因子框架

## 统一方向

本框架统一采用“因子值越大，预期收益越高”的方向：

- `volume_price_alpha = -corr(adjusted_close, turnover, 10)`
- `turnover_cv_stability = -std(turnover, 20) / mean(turnover, 20)`
- `lottery_avoidance = -max(daily_return, 21)` 或 `-average(topN_daily_return, 21)`

## 因子 1：量价相关性

- 数据：复权收盘价、日换手率。
- 窗口：半个月，默认 10 个交易日。
- 原始定义：`corr(close, turnover)`。
- Alpha 方向：取负号。
- 行为拆解：
  - `volume_down`：量价背离 + 前期下跌，偏多头来源。
  - `volume_up`：量价同向 + 前期上涨，偏空头来源。
  - `shrinking_up`：量价背离 + 前期上涨，信号较弱。
  - `shrinking_down`：量价同向 + 前期下跌，信号较弱。
- 必做复核：市值、1 个月反转、3 个月反转、平均换手、行业中性化。
- 失效风险：增量资金牛市、题材持续扩散、流动性急剧宽松。

## 因子 2：换手率变异系数

- 数据：日换手率。
- 窗口：20 个交易日。
- 原始定义：`CVTOR = std(turnover) / mean(turnover)`。
- Alpha 方向：取负号。
- 经济含义：交易行为不稳定性、流动性水平二阶矩、短期分歧和冲击交易。
- 样本过滤：上市满 12 个月；过去 20 日至少 15 日交易；信号日未停牌。
- 推荐持有期：5、10、15 个交易日优先；20 日需要单独复核。
- 必做复核：平均换手率、市值、20 日涨跌幅、BM、停牌和极端成交。

## 因子 3：博彩型股票规避

- 数据：日收益率。
- 窗口：过去一个月，默认 21 个交易日。
- 原始定义：`max_daily_return` 或 `average(topN_daily_return)`。
- Alpha 方向：取负号。
- 行为含义：彩票偏好、极端收益追逐、获利了结压力。
- N 值：N=1 表示单日极端收益；N=3/5 更能捕捉持续极端上涨簇。
- 必做复核：市值、短期反转、流动性缺乏度、BM、Beta、异质波动率。
- 组合用途：更适合作为空头规避、风险过滤或高波动主题降权，不应只用作多头增强。

## 因子 4：因子库分层与状态条件

- 月度行情因子优先于季度财务因子进入短周期模型。
- 每个因子必须记录：
  - 原始方向。
  - 统一 alpha 方向。
  - 预测周期。
  - 数据可得时点。
  - 风格暴露。
  - 市场状态依赖。
- 市场状态分层：
  - 上期市场上涨：多数行情因子区分度更强。
  - 上期市场下跌：反转、52 周高点、滞后动量等可能更有用。
- 中性化规则：
  - 先做市值中性，再看收益衰减。
  - 再做反转中性，区分独立 alpha 与反转代理。
  - 对短周期因子额外做平均换手和波动率中性。

## 研究质量门

- 原始分组、双变量分组、截面回归至少三种检验交叉验证。
- 每个因子输出原始 IC、中性化 IC、分组收益、多空收益、换手、成本后收益、容量估计。
- 对行为因子必须写清楚替代解释：反转、小市值、流动性、估值、波动率。
- 若因子主要贡献来自空头端，需要标记为“空头收益型”，不能直接假设多头可交易。
