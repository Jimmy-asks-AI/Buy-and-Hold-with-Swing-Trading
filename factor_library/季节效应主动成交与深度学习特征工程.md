# 因子卡：季节效应、主动成交度与深度学习特征工程

## 季节效应因子择时

用途：对已有因子或风格暴露做小幅择时，不作为独立 alpha。

规则原型：

- 2、3、5、8月：可小幅放松小盘暴露。
- 5、6月：可小幅放松成长/高估值暴露，并增强ROE/SUE权重。
- 下半年：可提高反转、低波、低换手因子权重。
- 12、1、5、6、9、10月：可增强基本面因子权重。
- 节前5日：偏高盈利、低换手、大盘蓝筹。
- 节后5日：偏小市值、高增长、反转弹性。

落地方式：

- 组合层：卫星组合开关。
- 风控层：按日历调整风格暴露上下限。
- alpha层：对因子权重做不超过基准权重的小幅倾斜。

代码：`strategy_lab/factor_timing_seasonality.py`

核心函数：

- `month_premium_summary`
- `month_bucket_regression`
- `holiday_window_summary`
- `seasonal_exposure_schedule`
- `apply_exposure_schedule`

## 小单主动成交度

用途：订单流高频因子，适合周频/月频选股和指数增强增量测试。

定义：

- 对逐笔成交按买单号/卖单号重构订单。
- 对买单，主动成交金额为 tick 方向为主动买入的金额；对卖单，主动成交金额为 tick 方向为主动卖出的金额。
- 主动成交度 = 主动成交金额 / 订单成交金额。
- 以个股20日订单成交额分布区分小/中/大单。

重点因子：

- 小买单主动成交度（全天）
- 小买单主动成交度（盘中）
- 小买单超额主动成交度
- 小卖单超额主动成交度

使用规则：

- 先对行业、市值、换手、反转正交。
- 检查与买入意愿、大单净买入、尾盘成交占比、深度学习高频因子的相关性和边际贡献。
- 基础模型没有深度学习高频因子时更有增量；已有深度学习高频因子时增量不稳定。

代码：`strategy_lab/order_flow_factors.py`

- `reconstructed_order_active_degrees`
- `order_active_degree_features`

## 深度学习高频特征工程

用途：将高频原始数据转为机器学习 alpha 输入特征，并降低过拟合风险。

流程：

1. 原始数据降频到分钟级基础指标。
2. 使用固定算子、变换输入的方式生成目标频率衍生特征。
3. 偏度调整，处理波动率、成交金额、成交笔数、买卖单数等右偏特征。
4. 去极值，通常用 N 倍标准差截断。
5. 标准化，优先横截面标准化；跨截面标准化作为备选。
6. 用积分梯度等方法做特征归因。
7. 用滚动历史贡献度做动态特征筛选，常用 Top128/Top64。
8. 回测前做因子归因、风格归因、交易成本、容量和拥挤度检查。

## 多颗粒度模型

推荐研究顺序：

1. 单颗粒度日频、60分钟、30分钟分别训练。
2. 输出集成作为第一优先级，因为简单、稳健、报告中效果较好。
3. 混合输入作为第二方案。
4. 双向AGRU可改善RNN遗忘问题，但需要严格样本外测试。
5. 残差学习网络复杂度高，复现未体现优势，暂不作为默认方案。

质量门槛：

- 标签长度必须匹配调仓周期和成交假设。
- 周频模型必须使用周频风险模型和周频成本估计。
- 多颗粒度信息增益要通过单颗粒度、混合输入、输出集成、残差模型横向比较。
- 需要报告不同选股域的表现，尤其是中证800内、中证1000、国证2000、成交额前20%股票池。
- 需要监控拥挤迁移：某些股票池历史强但近期显著衰减时，不能机械外推。

## 失效风险

- 日历效应公开后衰减或反转。
- 假日效应受市场制度、资金面和节前宏观事件影响。
- 主动成交度依赖交易所逐笔字段，数据商字段差异会导致不可复现。
- 高频特征容易受异常行情、涨跌停、停牌、流动性断层影响。
- 深度学习模型样本外衰减、特征泄露、成本低估、拥挤交易风险高。
