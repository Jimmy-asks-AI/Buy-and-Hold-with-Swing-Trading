# 条件期望因子择时

## 研究问题

在市场状态变化时，如何用量化方法动态调整因子权重，而不是主观加减仓某个因子。

## 输入

- 历史因子收益：可使用因子 IC、因子多空收益、回归 beta 或因子组合收益。
- 条件变量：市场涨跌幅、波动率、估值、换手率、利率、宏观变量等。
- 当前条件变量值：调仓时点已经可观察的状态。

## 模型

联合正态条件期望：

```text
E[f | v*] = mean(f) + Cov(f,v) * Inv(Cov(v,v)) * (v* - mean(v))
Cov[f | v*] = Cov(f,f) - Cov(f,v) * Inv(Cov(v,v)) * Cov(v,f)
```

动态因子权重：

```text
w ∝ Inv(Cov[f | v*]) * E[f | v*]
```

## 条件变量筛选

使用 AIC 前向筛选：

```text
AIC = T * log(|Cov[f | v*]|) + 2 * N * K
```

若加入候选变量不能降低 AIC，则停止加入，避免条件变量过多造成过拟合。

## 适用场景

- 多因子模型出现风格切换风险。
- 原始因子权重过度集中于市值、反转、低波、低换手等阶段性强因子。
- 需要在不完全推翻长期模型的前提下，提高对市场状态变化的反应速度。

## 不适用场景

- 因子本身没有稳定长期收益。
- 条件变量只有样本内解释力，没有滚动样本外验证。
- 用当前不可得的宏观或估值数据做调仓。
- 把因子择时误用为指数择时或仓位择时。

## 验证清单

- 因子收益和条件变量是否按可得日期对齐。
- 是否使用滚动窗口，并且当前预测只用过去数据估计协方差。
- 条件变量是否经过 AIC、样本外或稳定性筛选。
- 是否比较原始无条件模型和条件模型。
- 是否分风格切换期、趋势期、震荡期评估。
- 是否报告换手、成本、权重漂移和单因子权重上限。
