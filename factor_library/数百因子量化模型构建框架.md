# 数百因子量化模型构建框架

创建日期：2026-05-23  
对应代码：`strategy_lab/multi_factor_research_framework.py`

## 总体架构

```text
数据层 -> 因子注册表 -> 清洗标准化 -> 中性化 -> 单因子诊断
     -> 冗余聚类 -> 家族内合成 -> 家族间模型 -> 目标权重
     -> 回测执行 -> 绩效归因 -> 复习纠错 -> 入库/降级/淘汰
```

## 1. 因子注册表

每个因子必须有：

- `factor_id`
- `column`
- `family`
- `direction`
- `horizon`
- `data_type`
- `availability_col`
- `cost_tier`
- `description`

没有注册信息的因子不能进入模型。尤其是财报、分析师、公告、文本、另类数据，必须明确可得日期。

## 2. 因子家族

建议初始家族：

- `value`
- `quality`
- `profitability`
- `growth`
- `dividend`
- `investment`
- `leverage`
- `momentum`
- `reversal`
- `liquidity`
- `volatility`
- `technical`
- `intraday`
- `level2`
- `analyst`
- `fund_flow`
- `behavior`
- `macro`
- `alternative`
- `ml_deep`
- `llm_text`

家族的作用是防止同类因子重复投票。例如价值类中 EP、BP、SP、FCFP、无形资产调整 PB 不能简单当作 5 个完全独立因子。

## 3. 单因子检验

入库前最低诊断：

- 覆盖率。
- 缺失率。
- 原始 IC 和 RankIC。
- 中性化后 IC 和 RankIC。
- 分组收益和高低组差。
- 换手率。
- 行业、规模、流动性偏离。
- 分年度和分市场状态表现。
- 成本敏感性。
- 失败场景。

## 4. 冗余控制

数百因子必须做相关性聚类：

1. 计算每期横截面因子相关性。
2. 对时间求平均相关矩阵。
3. 用绝对相关阈值形成连接图。
4. 每个高相关簇中只保留质量分最高的因子，或先合成簇内因子。

推荐阈值：

- 初筛：`abs(corr) >= 0.85`
- 严格：`abs(corr) >= 0.75`
- 同家族内部可更严格，跨家族可更宽松。

## 5. 因子质量分

质量分不应只看 IC 均值。建议至少包含：

```text
quality_score =
    RankIC均值
    * sqrt(有效期数)
    * (RankIC胜率 + 0.5)
    / (1 + 平均换手)
```

可扩展项：

- 覆盖率惩罚。
- 成本惩罚。
- 容量惩罚。
- 分年度稳定性惩罚。
- PBO 惩罚。
- 过度拥挤惩罚。

## 6. 家族内合成

每个家族内部先做合成，合成方式按复杂度递增：

1. 等权 z-score。
2. 滚动 RankIC 加权。
3. 滚动 ICIR 加权。
4. 稳健 PCA。
5. ElasticNet / Ridge。
6. LightGBM / TabNet / 深度时序模型。

默认从简单模型开始，只有当复杂模型带来样本外增量时才升级。

## 7. 家族间模型

家族间合成应更保守：

- 价值、质量、红利、成长等基本面家族适合低频权重。
- 动量、反转、技术、Level2 等家族适合更短半衰期。
- 宏观和状态因子更适合作为权重调节器，而不是直接 alpha。
- AI/深度因子必须先证明正交后增量。

## 8. 组合权重

推荐接口：模型输出目标权重，而不是直接输出订单。

权重约束：

- 单票最大权重。
- 行业偏离。
- 风格暴露。
- 流动性约束。
- 换手约束。
- 停牌、涨跌停、成交额过滤。
- 交易成本和冲击成本。

## 9. 回测与验收

验收必须分三层：

### 研究层

- 因子 IC。
- 分组收益。
- 覆盖率。
- 换手。
- 冗余。

### 组合层

- 年化收益。
- 超额收益。
- 信息比率。
- 最大回撤。
- Calmar。
- Sortino。
- 月度胜率。
- 成本后收益。

### 生产层

- 订单模拟。
- 成交价假设。
- 冲击成本。
- 容量。
- 实盘漂移。
- 因子延迟和数据修订。

## 结论

有能力构建数百因子模型，不等于要一次性使用数百个因子。正确做法是维护数百个候选因子，但每次实盘模型只让通过质量门槛、冗余控制和样本外检验的一部分因子进入有效模型。
