# 华泰标准单因子测试框架

## 适用范围

用于股票截面因子的基础有效性验证，尤其适合估值、成长、质量、动量、反转、流动性、技术指标等月频或周频因子。它回答的问题是：一个因子是否稳定解释未来横截面收益。

它不能单独回答：

- 策略实盘收益是否可实现。
- 组合换手和冲击成本是否可承受。
- 因子未来是否仍然有效。
- 多个因子合成后是否仍有独立增量。

## 数据结构

最小输入字段：

- `date`: 截面日期或调仓日。
- `asset`: 股票代码。
- `factor`: 因子原始值。
- `forward_return`: 下一期收益评价标签。
- `industry`: 行业分类，可选但强烈建议。
- `sqrt_float_mkt_cap`: WLS 权重，可选。
- `size_control`: 市值、对数市值或其他控制变量，可选。

财务类因子必须按公告可得日对齐，不能直接使用报告期数据。

## 标准步骤

1. 方向处理：把原始因子转换为“值越大预期收益越高”的 alpha score。
2. 去极值：每期按中位数和 MAD 截断。
3. 标准化：每期 z-score。
4. 缺失处理：标准化后可填 0 作为中性暴露，但必须保留覆盖率报告。
5. 截面回归：下期收益对因子、行业哑变量和控制变量做 WLS。
6. IC 检验：计算 Pearson IC 与 Rank IC。
7. 分层组合：按因子分位数组合，看高低组差、单调性和稳定性。
8. 汇总判断：结合回归 t、因子收益时间序列 t、IC IR、胜率、分层单调性。

## 因子方向约定

正向原始因子：

- `EP`、`EPcut`、`BP`、`SP`、`NCFP`、`OCFP`、`FCFP`、`DP`
- `Sales_G_q`、`Profit_G_q`、`ROE_G_q` 等成长增速

反向原始因子：

- `EV2EBITDA`、`PEG`
- `return_1m`、`wgt_return_1m`、`exp_wgt_return_3m`、`exp_wgt_return_6m` 等 A 股反转类过去收益指标

正向 alpha score 统一为：

```text
alpha_score = raw_factor * expected_alpha_direction
```

其中 `expected_alpha_direction` 为 `1` 或 `-1`。

## 核心判据

一个因子进入候选库前至少应满足：

- 有经济含义或行为金融解释。
- 数据定义清晰，且可得时间不含未来信息。
- Rank IC 或回归因子收益在滚动窗口中方向稳定。
- 分层收益有基本单调性，或高低组差有可解释性。
- 与已有核心因子不是完全重复。
- 扣除交易成本后仍可能保留收益。
- 在不同行业、市值、市场阶段中没有明显集中失效。

## 失败模式

- 行业偏差：因子只是在买入某些行业。
- 市值伪暴露：因子有效性来自小市值溢价。
- 财报未来函数：使用了公告日前不可得数据。
- 多重检验：大量因子和参数中挑出偶然赢家。
- 方向误设：低值更优的因子未反向，导致合成时抵消收益。
- 缺失伪中性：缺失覆盖率过高但全部填 0。
- 成本吞噬：高换手因子的纸面收益不可实现。
