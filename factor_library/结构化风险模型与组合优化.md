# 结构化风险模型与组合优化

## Purpose

为中高频 alpha、机器学习模型和指数增强组合提供可验证的风险预测与组合约束框架。

## Core Formula

```text
stock_cov = X * factor_cov * X' + specific_var
predicted_vol = sqrt(w' * stock_cov * w)
active_objective = alpha' * active_weight - lambda * active_weight' * stock_cov * active_weight
```

## Risk Model Checklist

- 因子暴露去极值、缺失值填充、标准化。
- 必要但不过度的因子正交。
- WLS 估计因子收益和特异性收益。
- EWMA 估计协方差。
- Newey-West 调整收益自相关。
- 特征值或 PSD 修正，避免不可逆或非半正定。
- 偏误统计量验证预测风险。
- 预测期限必须匹配调仓周期。

## Bias Statistic

```text
b_t = realized_return_t / predicted_vol_t
B = std(b_t)
CI = [1 - sqrt(2/N), 1 + sqrt(2/N)]
```

- `B > CI_high`: 风险低估。
- `B < CI_low`: 风险高估。
- `B` 落入置信区间：预测风险相对合理。

## Optimization Rules

- `lambda` 越高，风险惩罚越强，跟踪误差通常越低，超额收益通常越低。
- 行业、市值、风格中性会降低风险暴露，但也会压缩 alpha 表达。
- 放松行业偏配可能提高收益，但会增加跟踪误差和回撤。
- 股票池越宽，alpha 表达空间越大；只在成分股内选股通常降低超额收益。
- 换手约束必须与信号半衰期匹配。

## Failure Modes

- 用月频风险模型管理周频组合。
- 只做收益回测，不验证风险预测偏误。
- 风险模型因子缺失或过多导致估计不稳定。
- 机器学习 alpha 实际暴露到风格/行业收益，误判为纯 alpha。
- 约束过多导致目标函数难以表达 alpha。
- 约束过少导致组合过度集中。
