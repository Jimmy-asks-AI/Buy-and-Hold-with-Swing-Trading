# 行为准另类技术择时因子框架

创建日期：2026-05-23  
对应代码：`strategy_lab/behavior_altdata_technical_timing_models.py`

## 1. 融资增速因子

**定义**

```text
financing_delta = margin_financing_balance_t - margin_financing_balance_{t-k}
financing_growth = rolling_mean(daily financing_delta, k) / rolling_mean(amount, k)
```

**方向**

融资增长越强，短中期买盘压力越强。方向默认正向，但在过热、监管收紧或高拥挤时需要降权。

**关键参数**

- 观察窗口：20、40、60 日。
- 持有期：20、30、60 日。
- 横截面处理：行业、市值、成交额、换手率中性化。

**失效场景**

- 融资融券标的扩容期。
- 牛市后期高杠杆拥挤。
- 成交额急剧萎缩导致分母异常。

## 2. 融券警告因子

**定义**

```text
short_growth = rolling_change(short_balance, k) / rolling_mean(amount, k)
```

**用途**

不默认作为做空 alpha，仅作为风险警告或融资信号的折减项。A 股融券余额受券源限制明显，不能等同于真实看空力量。

## 3. 并联极端因子组合

**定义**

对若干单因子分别进行横截面分位筛选，选择每个因子的极端头部股票，再合并成组合。

**推荐输入**

- 市值
- 成交额
- 一个月反转
- 换手率
- 估值

**复核项**

- 单因子极端组收益差。
- 被选股票未来收益分位分布。
- 因子之间持仓重叠度。
- 行业、市值和流动性偏离。
- 成本后净收益。

## 4. 应付薪酬增长因子

**定义**

```text
payable_comp_growth_yoy = payable_comp_t / payable_comp_{t-4Q} - 1
```

可扩展为人均口径、行业中性口径和连续增长口径。

**方向**

默认正向，代表人力资本投入、业务扩张或管理层对未来需求的信心。

**必要约束**

- 使用财报公告日或数据入库日。
- 同季度同比，避免季节性错配。
- 行业内标准化。
- 与经营现金流、收入增长和研发投入联动复核。

## 5. 关系公司动量

**定义**

```text
related_momentum_i = sum_j weight_{i,j} * return_j
```

其中关系权重可来自专利相似、地理邻近、共同客户、共同分析师覆盖、供应链关系等。

**经济逻辑**

市场可能低估关联公司信息向目标公司的传导速度。

**风险控制**

必须剥离行业动量、市场动量、规模、流动性和分析师覆盖共同暴露。

## 6. 基金隐含 alpha

**定义**

```text
fund_implied_alpha_stock = sum_f holding_weight_{f,stock} * fund_skill_f * flow_or_adjustment_f
```

**用途**

识别有技能资金的隐含偏好，而不是简单追踪基金规模流入。

**复核项**

- 持仓披露滞后。
- 基金经理稳定性。
- 基金风格漂移。
- 流入是否只是追涨。
- 因子衰减期限。

## 7. 技术指标类别投票择时

**定义**

先把多个技术指标映射为 `-1/0/1`，再按类别聚合：

```text
category_vote = sign(mean(indicator_signals_in_category))
final_position = count(category_vote > 0) / category_count
```

**类别**

- 价格动量
- 价格反转
- 成交量
- 价量结合
- 市场宽度

**使用方式**

适合作为指数、ETF 或底仓的仓位调节器，不宜默认每日全仓多空切换。

**成本控制**

- 降低调仓频率，如 5 日或 10 日调仓。
- 设置最小仓位变化阈值。
- 用交易成本后收益评价，而不是原始收益。

## 入库质量门槛

- 明确数据可得日期。
- 至少跨牛市、熊市和震荡市测试。
- 输出原始 IC、正交 IC、分组收益、换手、成本后收益。
- 报告容量和流动性约束。
- 报告失败年份和参数敏感性。
- 对行为解释提供竞争假设，不能只讲故事。
