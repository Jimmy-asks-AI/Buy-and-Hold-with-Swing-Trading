# 交易约束、纸面跟踪与真实数据 Panel 层

日期：2026-05-23

## 定位

因子工厂不能只停在预测层。一个可治理的多因子模型至少需要四个边界层：

1. 数据边界：所有字段必须有可得日期。
2. 执行边界：目标权重必须受交易状态、成交额和资金规模约束。
3. 纸面边界：实验晋级只进入纸面跟踪，不直接进入实盘。
4. 监控边界：纸面跟踪需要记录目标权重、状态、漂移检查项和人工复核项。

## Point-in-time Panel 原则

- 行情字段可以按交易日期直接使用。
- 财务字段必须使用公告日、入库日或可得日期，不能使用报告期末日回填。
- 行业、指数成分、ST、停牌、涨跌停等状态字段也必须按可得日期处理。
- 合并时必须按 `asset` 分组做 backward as-of，避免跨股票污染。

## 执行约束原则

- 不可交易资产权重为零。
- 单资产目标权重不能超过成交额容量上限。
- 容量约束后必须重新归一化。
- 若可交易池不足，实验应降级为 `revise_and_rerun` 或 `reject_until_data_fixed`。

## 纸面跟踪原则

- `promote_to_paper` 是工程和统计闸门通过，不是实盘授权。
- 纸面状态必须显式保存 `live_trading_allowed=false`。
- 纸面期至少观察目标权重稳定性、换手、容量占用、净值漂移、回撤和异常持仓。
- 纸面期出现数据延迟、容量超限、因子覆盖骤降或实盘不可成交，应回到研究阶段。

## 对多因子模型的影响

交易约束和纸面跟踪会改变模型选择标准。高 IC 但高换手、高容量占用、低覆盖或频繁不可交易的因子，不能因为回测收益高就进入生产组合。正式排序应使用净收益、容量调整收益、稳定性和治理评分共同决定。
