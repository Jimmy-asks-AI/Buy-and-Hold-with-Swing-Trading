# 行业质量、五因子定价与回归树择时框架

## 模型卡 1：行业内逐步多因子

- 目标：在特定行业内筛选有边际解释力的因子。
- 输入：行业股票池、次月收益、候选因子暴露。
- 方法：Fama-MacBeth 横截面回归；新增因子和已选因子溢价均显著时，选择调整 R2 增幅最大的因子。
- 适用：医药、机械、化工等股票数量足够的行业。
- 风险：行业样本窄、因子方向阶段性反转、行业分类漂移、容量不足。
- 代码：`industry_quality_factor_models.industry_stepwise_factor_selection`。

## 因子卡 1：质量复合因子

- 类型：基本面质量、多信号因子。
- 组件：盈利能力、成长、盈利稳定性、投资、净发行、资本结构。
- A 股修正：
  - 盈利能力为核心正向组件。
  - 投资因子按市值分层处理，不能全市场固定方向。
  - 净发行方向为负，但长持有期会衰减。
  - 杠杆变化方向为正，且在高盈利股票中更强；杠杆水平不宜单独入库。
- 代码：`quality_composite_score`、`net_share_issuance_factor`、`asset_growth_factor`、`leverage_change_factor`。

## 模型卡 2：A 股五因子定价

- 目标：作为 A 股事件、异象、策略和基金归因的基准模型。
- 因子：`Mkt`、`SMB`、`FPE`、`FSUE`、`FTurn`。
- 构建：2x3 市值和特征交叉分组；PE 与换手率默认低值优，SUE 默认高值优。
- 用途：解释策略收益来源，检验 alpha 是否独立，筛选冗余因子。
- 风险：因子定义依赖样本期，PE/SUE 优势需滚动验证，模型误设会误判 alpha。
- 代码：`two_by_three_factor_returns`、`factor_model_regression`、`factor_return_attribution`、`bic_model_posterior`。

## 模型卡 3：回归树防御性因子择时

- 目标：用非线性情景划分改善因子敞口管理。
- 输入：因子收益、债市指标、股市指标、因子拥挤指标。
- 核心规则：
  1. 用滚动窗口训练回归树预测因子收益。
  2. 用因子收益动量作为稳健基准。
  3. 若回归树与动量方向一致，使用回归树预测；否则因子预测设为 0。
- 适用：多因子权重分配、风格暴露风控、因子失效期防御。
- 风险：单棵树不稳定、阈值过拟合、频繁开关导致收益损失。
- 代码：`rolling_tree_factor_timing`、`factor_return_momentum_prediction`、`defensive_timing_signal`。

## 入库标准

1. 行业模型必须报告行业股票数量、因子覆盖率、每期有效样本和行业基准。
2. 质量因子必须拆分组件贡献，不能只报告复合分数。
3. 五因子归因必须先于新异象入库。
4. 回归树择时必须滚动样本外，禁止用全样本树解释历史。
5. 防御性择时要同时报告收益下降、回撤改善和敞口关闭比例。
