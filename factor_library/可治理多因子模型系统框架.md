# 可治理多因子模型系统框架

创建日期：2026-05-23

## 系统结构

```text
数据契约
  -> 因子计算器
  -> 因子注册表
  -> 注册表审计
  -> Panel 校验
  -> Walk-forward 训练/测试
  -> 因子筛选
  -> 家族合成
  -> 目标权重
  -> 成本扣减
  -> 绩效报告
  -> 实验账本
  -> 晋级/返工/拒绝
```

## 模型治理原则

1. 不允许全样本选因子后再宣称样本外。
2. 不允许缺可得日期的数据进入模型。
3. 不允许只看 gross return，必须看 net return。
4. 不允许只看 Sharpe，必须同时看回撤、胜率、换手、成本和覆盖率。
5. 不允许因子家族重复投票，必须先做冗余聚类和家族内合成。
6. 不允许 demo 结果被解释为真实 alpha。
7. 不允许没有账本记录的实验进入下一阶段。

## 晋级规则

当前实验账本使用：

- `promote_to_paper`
- `revise_and_rerun`
- `reject_until_data_fixed`

真实模型进入 `promote_to_paper` 后，只代表可以纸面跟踪，不代表可以实盘。

## 真实模型最低门槛

进入纸面跟踪前至少满足：

- 无 point-in-time 违规。
- walk-forward 净收益为正。
- 至少 3 个以上家族贡献。
- 不依赖单个因子或单个年份。
- 成本后 Sharpe 和 Sortino 达标。
- 最大回撤在可接受范围内。
- 因子覆盖率和股票覆盖率稳定。

进入实盘前还需：

- 真实交易成本。
- 容量约束。
- 停牌/涨跌停约束。
- 行业/风格约束。
- 纸面交易 3-6 个月。
